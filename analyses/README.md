## 1. マクロ・トリガー戦略

**仮説**: 日本市場寄付前の外部指標 (LME銅・原油・先物夜間) が日本株の寄引きを予測するか。

|---------|--------|---------|:-:|
| [20260421_lme_copper_overnight](20260421_lme_copper_overnight/) | **LME銅+1% → コア5 Long** | 本命戦略。月1-2回、1回+147bps/日 | **+11.50** |
| [20260422_lme_down_short](20260422_lme_down_short/) | LME銅-1% → コア5 Short | 逆方向でも有効だが発生N=13と稀少 | +5.51 |

### 主要発見

**A. LME銅オーバーナイトは最強のシグナル** — JST 3:00頃に確定するロンドン終値が、9:00寄付の方向を有意に予測。Sharpe+11.5は他戦略の2倍以上。

**B. 累積モメンタム > 単日モメンタム** — LME累積+3% (LB=10) が N=37で Sharpe+7.5。単日より発生頻度が高く使い勝手良し。

**C. 先物夜間シグナルは独立して効く** — LMEシグナルなしの日もカバー。ポートフォリオ重ね掛けに適する。

## 2. 曜日・レジーム分析

**仮説**: 日本株の曜日効果は戦争・マクロイベントで構造的に変わる。

| フォルダ | テーマ | 主要発見 |
|---------|--------|---------|
| [20260422_dayofweek_lme](20260422_dayofweek_lme/) | LMEアップ日の曜日別成績 | **木曜はSharpe-1.31で除外推奨**。火曜+13.15が最強 |
| [20260422_dayofweek_regime](20260422_dayofweek_regime/) | 2025-06 イスラエル/イラン12日戦争の前後比較 | レジーム別曜日効果の検証 |
| [20260422_dayofweek_iran2026](20260422_dayofweek_iran2026/) | **2026-02-28 イラン戦争**の前後比較 | 戦争期間-792bps、最弱日3/9月曜-380bps |
| [20260422_dayofweek_nonfer_semi](20260422_dayofweek_nonfer_semi/) | **非鉄 vs 半導体** 曜日比較 | 半導体**水曜全日+332bps (t=+3.85)**、ローム戦争中+1538bps逆行高、5713 住友金属-25%DD |

### 主要発見

**A. 木曜は鬼門** — LMEアップ日でも木曜エントリーはSharpe-1.31。全戦略に共通する警戒曜日。

**B. 2026-02イラン戦争で非鉄/半導体の挙動が分岐**
- 非鉄: ON強い / 日中弱い (5713 住友金属鉱山 -25% DD)
- 半導体: 水曜全日でパフォーマンス集中 (+332bps, t+3.85)
- **6963 ローム**のみ戦争中+1538bps (セーフヘイブン的動き)

**C. Post期間 N=25日で曜日あたり4-6サンプル** — 過学習リスク極大。t-stat と N をセットで評価すべき。

## 4. イントラデイ構造分析

**仮説**: 時間帯別の典型パターン (ボラU字型・ギャップフィル・寄付30分効果) が戦略設計に活かせる。

| フォルダ | テーマ | 主要発見 |
|---------|--------|---------|
| [20260421_semiconductor_leadlag](20260421_semiconductor_leadlag/) | 半導体リードラグ | 銘柄間の先行遅行関係 |
| [20260422_nonfer_intraday_patterns](20260422_nonfer_intraday_patterns/) | **非鉄3銘柄** イントラデイ典型パターン | 5706 first30 reversal でコスト後+10bps/日が最有力 |
| [20260422_semi_intraday_patterns](20260422_semi_intraday_patterns/) | **半導体5銘柄** イントラデイ典型パターン | **6857 first30<-50 Long (WR 65%, +26bps)** / **4063 first30>+50 Long (+31bps)** |
| [20260422_orb_morning](20260422_orb_morning/) | ORB (9:00-9:30 → 11:30) | 全体Sharpe+0.3、銘柄依存でエッジ薄い |
| [20260422_orb_volume](20260422_orb_volume/) | **出来高フィルター付きORB 最適化** | 集約では効果なし。**5711 Sharpe+5.83 / 4063 Sharpe+6.04** が合格 (方向は銘柄特性で決定) |
| [20260422_gap_fade](20260422_gap_fade/) | ギャップフェード/モメンタム | 全閾値で Sharpe<1、明確にエッジなし (不採用) |
| [20260422_vwap_direction](20260422_vwap_direction/) | **VWAP乖離方向判定** | 銘柄別エッジ: 8035 11:30 Reversion Sharpe+6.17 / 4063 10:00 Trend Sharpe+2.95。Pool集約はフラット |

### 主要発見

**A. U字型ボラは全銘柄共通** — 9:00分足 80-120bps、11:30で 4-6bps、12:30で再度上昇。11:30は指値執行に最適。

**B. 高値/安値の約45%が 9:00-9:30 で形成** — OR (Opening Range) を基準にしたブレイクアウトの土台。

**C. 非鉄 vs 半導体の性格差**
| 項目 | 非鉄 | 半導体 |
|---|---|---|
| EODドリフト | +27bps | +20bps |
| first30→rest corr | 弱 (\|<0.05\|) | **4063 +0.14 (唯一のモメンタム)** |
| ギャップフィル率(down) | 54% | 49% |
| 最有力シグナル | 5706 first30 reversal | **6857 first30<-50 Long (WR 65%)** |

**D. 6857 アドバンテスト**: first30<-50bps → rest +30bps (WR 65%) が全28銘柄中最強の単一シグナル。

**E. VWAP乖離は11:30で最強のReversion時刻** — 8035 TEL `|dev|>=120bps` → 15:30決済で Sharpe+6.17 (t=+2.09, WR 69%)。4063 信越化学は全時刻でTrend型 (first30/ORBと整合)。Pool集約はフラットで、方向は銘柄特性 (mom/rev) で決定する必要あり — 3分析 (first30/ORB/VWAP) で同一結論。

```python
df['jst'] = df['timestamp'] + pd.Timedelta(hours=9)
```
