# ティック由来 新規シグナル広範探索

## 目的

これまでに検証していない、ティック由来の新シグナル群を一括テスト。  
B-Optimized (Sharpe=18) と別の独立したエッジを見つけたい。

---

## データ概要

- 対象: SBG (99840) のみ (流動性 + 既存データ)
- 期間: 2026-01-01 〜 2026-04-30 (4ヶ月、学習期と同じ)
- ベース: ティック 5,437,502件 → 1分バー 23,544件
- 各バーに**10種類以上の新特徴量**を付与

### 新規特徴量

| カテゴリー | 特徴量 | 意味 |
|---|---|---|
| **OFI** | `ofi`, `cum_ofi_5b`, `cum_ofi_15b` | uptick vol - downtick vol の不均衡度 |
| **TradeSize** | `avg_trade_size`, `_ratio` | 1約定の平均株数とその急増度 |

| **VolExpand** | `range_ratio` | レンジの急拡大 |
| **Concentration** | `concentration` | 単発が出来高の何%か |

## 解釈

### 発見1: 「大口プリント"群"」は単発より強い

これまでの B-Optimized は**単発大口プリント**を見ていたが、今回見つかった D2/D3 は **1分内に複数の大口プリントが集中する**現象を捉えている。

```
D2: 1分間に大口プリント (≥8200株) が5件以上同時発生
    → 30分後に+12bps平均 (+8bps net)
D3: 直近5分で累積10件以上の大口プリント
    → 30分後に+9bps平均 (+5bps net)
```

意味: **機関の「波状買い」(間隔をあけずに連続発注) は本物のフロー**で、続伸する。  
逆に「**まばらな単発大口**」はノイズ・在庫調整の可能性が混じる。

### 発見2: 短期モメンタム継続 (I1) も機能

```
I1: 直近15分で+50bps以上上昇 → 30分後も+9bps平均 (+5bps net)
```

これは「**機関買いの初動 → 中期継続**」を捉えている。B-Optimized と発想は近いが、シグナル定義が異なる。

### 発見3: OFI (Order Flow Imbalance) は機能しない

事前にtop候補だと思ったOFI (上昇ティック vol vs 下降ティック vol の差)は**全て非有意**:

```
A1: OFI≥0.3 ロング 30分 → t=0.35, 非有意
A2: OFI≤-0.3 ショート 30分 → t=-0.22, 非有意
J1: OFI≥0.3 × 出来高≥2x → t=-0.44, 非有意
K1: 寄付30分 × OFI≥0.3 → 非有意
```

理由: OFI は1分粒度では**ノイズが多すぎる**。大口プリントの方向性 (B戦略) や、大口プリント"群"の検出 (D系) のほうがシグナル品質が高い。

### 発見4: 平均回帰 (Mean Reversion) は機能しない

```
H1: 15分で+50bps上昇 → 反転売り: t=-4.80 (むしろ強い順方向継続)
H2: 15分で-50bps下落 → 反転買い: t=1.76, 非有意
H3/H4: 5分スケールも同様
```

SBG では**短期 (5-15min) は明確にモメンタム支配**。反転狙いは効かない。

### 発見5: その他の没アイデア

- **TickSpeed (C系)**: ティック頻度の急増は方向予測に貢献せず
- **VolExpand (F系)**: レンジ急拡大は方向不明 (上にも下にも振れる)
- **Concentration (G系)**: 単発の占める比率は B戦略に近いがエッジ薄い

## 既存戦略との関係

### B-Optimized vs D2/D3 の違い

| 項目 | B-Optimized | D2/D3 (新発見) |
|---|---|---|
| シグナル単位 | 単発大口プリント (uptick) | 1分内の大口プリント数 |
| 必要条件 | 当該tickが>=8200株かつuptick | 1分内に5件以上 大口プリント |
| エントリー時刻 | tick発生直後 | 1分バー終了時 |
| 保有時間 | 60分 | 30分 |
| Sharpe | ~18 (4ヶ月学習) | 5-8 |
| **特性** | 高 Sharpe だが 5/11 で大損 | 中程度 Sharpe、レジーム耐性は別途検証 |

### D系の利点

1. **シグナル頻度が安定**: 単発大口は出来高変動でブレやすいが、複数件カウントは安定
2. **計算が簡単**: 1分バー終了時に判定すれば良い (ティックレベルの瞬間判定不要)
3. **レイテンシ耐性**: 1分後の判定なので秒単位の遅延に強い
