# 信用残データ駆動 週次クロスセクショナル戦略 (Short Squeeze系)

## 結論を先に

**🚫 信用残データだけでは、コスト後実用レベルのエッジは見つからず。**  
ただし**直近2025-2026 では一部シグナルが機能**しており、レジーム依存的に有効化する可能性。

---

## 仮説と検証

完全に角度を変え、ティック解析ではなく**週次の需給データ**を使った戦略を検証。  
データ: `jquants_margin_interest` (信用残, 週次金曜公表), 2021-2026 5年分

| 仮説 | 内容 |
|---|---|
| H1 | 信用買い残 急増 → 投機過熱 → 翌週反落 (空売り対象) |
| H2 | 信用売り残 急増 → 踏み上げ → 翌週上昇 (Short Squeeze ロング) |
| H3 | 信用倍率 (long/short) 高水準 → 過熱 → 反落 |
| H4 | 信用倍率 低水準 (空売り過多) → 踏み上げ → ロング |
| H5 | Z-score 異常値ベースのシグナル |
| H6 | Long-Short 両建てポートフォリオ |

---

## データ概要

- 対象: 流動性Top **462銘柄** (信用取引対象 + 出来高 >100万)
- 期間: 2021-01-08 〜 2026-05-01 (262週)
- 観測数: 113,347 (週 × 銘柄)
- 翌5営業日リターン (市場ニュートラル化済)
- コスト: 両建てで往復8bps

## 主要結果

### 全期間 (5年)

| シグナル | N | mean (bps) | t値 | 勝率 | Sharpe |
|---|---|---|---|---|---|
| H4: 信用倍率減 Bottom5% ロング | 5,410 | +11.7 | 0.83 | 45.1% | 0.08 |
| H2: 信用売増Top20% ロング | 19,982 | +4.2 | 0.76 | 45.9% | 0.04 |
| H3: 信用倍率増Top10% ショート | 11,165 | +4.4 | 0.69 | 52.8% | 0.05 |
| H1: 信用買増Top5% ショート | 5,750 | +7.1 | 0.66 | 53.9% | 0.06 |
| ... 大半が t<1 で非有意 ... | | | | | |

**ベスト Sharpe = 0.08** — コスト無視でも実用には程遠い。

### 年別ロバストネス: 完全にレジーム依存

| 年 | H4: 空売り過多→ロング | H1: 信用買増→ショート | H2: 信用売増→ロング |
|---|---|---|---|
| 2021 | **+33.5bps** ✓ | +2.6 | +25.7 |
| **2022** | **-29.9** ✗✗ | +10.0 | **-31.7** ✗ |
| 2023 | +3.0 | -16.8 | -1.3 |
| 2024 | -21.2 | +15.1 | +5.1 |
| 2025 | -2.7 | -26.7 | +12.6 |
| **2026** | **+60.2** ✓ | -28.7 | **+68.0** ✓ |

→ **2021年と2026年は機能、2022年は完全に逆**。  
方向性が年によって反転 → 戦略として安定しない。


## なぜ機能しなかったか

### 1. 日本の信用残データの性質

日本市場の信用取引は**個人投資家中心** (制度信用は法人禁止枠が多い)。  
個人の動向は機関の先行指標にはならない (むしろ contrarian な動きが多いが、それも安定しない)。

### 2. シグナルの遅さ

- データ公表: 毎週金曜 (引け後)
- 翌週月曜から実行 → 既に織り込まれている可能性
- 機関は **証券金融会社の貸借残** (リアルタイム) を見ているため、週次データは遅い

### 3. クロスセクショナル効果が薄い

各週で 433銘柄しかなく、上位/下位10% でも 43銘柄程度。Sharpe を稼ぐには分散効果が不足。

---

## ただし: 直近2026年の挙動は注目

**2026年だけ見ると、H4 (空売り過多→ロング) と H2 (信用売増→ロング) が機能**:

| 戦略 | 2026年 N | 2026 mean | 2026 勝率 |
|---|---|---|---|
| H4: 空売り過多 → ロング | 659 | +60.2 | 44.5% |
| H2: 信用売増 → ロング | 606 | +68.0 | 45.0% |

これは「**直近の上昇相場では短期売り方が踏み上げられやすい**」というレジーム特性。  
ただしt値1.1〜1.2 と有意水準には届かず、4ヶ月のサンプルなので偶然の可能性も高い。

---

## 限界・学んだこと

1. **信用残データ単独では弱い**: シンプルな急変動シグナルでは Sharpe 0以下
2. **レジーム依存性**: 2022年と2026年で方向反転 → 体系的なエッジなし
3. **公表遅延**: 週次データは既に織り込み済の可能性高い
4. **個人投資家データの限界**: 機関投資家のフローを反映しない

## 検証してみるべき次の角度

信用残単独は薄かったので、**他のデータと組み合わせる**か、**別データを使う**:

### 案 A: 投資部門別売買データ
- `investor_types` テーブル (海外勢・個人・機関の売買フロー、週次)
- **海外勢が大幅買い越し → 翌週も継続バイアス** という古典的効果
- データ: 10年分あり

### 案 B: 機関の個別空売り報告
- `jquants_short_sale_report` (公開大量保有報告)
- 個別の機関 (Citadel, Marshall Wace, Maverick 等) の空売り建玉
- **特定機関の空売り集中 → 同銘柄の翌月リターン低下** という研究あり
- データ: 893K rows, 3,716銘柄

### 案 C: イベント駆動
- 信用規制発動日 (`jquants_margin_alert`)
- 制限解除後の踏み上げ効果

### 案 D: マクロ × 個別株
- USDJPY × 輸出株 (トヨタ・任天堂等)
- 米10年金利 × 銀行株
- VIX × 高ベータ銘柄

### 案 E: 日経先物 (JNIc1) リードラグ
- archive.intraday_data に2024-2026のJNIc1 1分足あり
- 個別株とのリードラグ (先物が5分早く動く?)

### 案 F: ETF裁定
- 日経225 ETF (1321等) と現物バスケットの瞬間乖離

---

## 推奨次手

個人的に**最も有望そう**なのは **案 A (投資部門別売買データ)**:
- データが10年分あり統計的に強い
- 海外勢の動向は機関主導なので、個人(信用)とは異なる先行性が期待できる
- 既存DBですぐ実装可能

または **案 B (機関の個別空売り報告)**:
- 「Citadel が SBG を空売り増 → 翌月下落」のような特定機関のフロー追随
- データもDB完備
