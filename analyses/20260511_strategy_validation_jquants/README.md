# 採用7戦略 JQuantsベース継続性検証 (2026-05-11)

## 目的

データソース移行 (Refinitiv→JQuants) 後の新DB (`stocks_intraday`/`stocks_daily`/`index_daily`) で、
書き換え済みの全7採用戦略が引き続き機能するかを実データで再検証。

前回 (2026-05-09) は archive (Refinitiv凍結, 1年) ベースだったため、今回は **JQuants最大期間 (イントラ2年・日足5年)** で再評価することで真の継続性を見る。

- **検証期間**: イントラ系=2024-05〜2026-05 / 日足系=2021-05〜2026-05

## 主要発見

### ❌ topix_overnight: 期間延長で機能消失 (Sharpe +6.27 → +0.58)
- N=363 (5年) で Sharpe +0.58、平均+5.8bps、WR 55.6%
- **前回 N=75 (約1年) の Sharpe +6.27 は特定期間 (2025-04〜2026-05) のみの現象**
- 2021-2024年は機能していなかった可能性 → 過学習の典型
- **対応**: 戦略停止または時期限定 (例「特定マクロ局面のみ発動」のフィルタ追加) 検討

### ❌ pair_portfolio: 18ペア EW では勝ち負け相殺
- 全体 Sharpe +0.65 (前回 +1.37 から低下)
- ペア別では半数が勝ち、半数が負け、相殺している
- **対応**: 勝ち7ペアのみに絞る、もしくは ペア別 Sharpe で動的ウェイト
- **半TEL-レーザー Sharpe +12.89** は単独でも採用価値あり

### ⚠️ lasertec_ma25_support: 期間拡張で平均化 (Sharpe +7.57 → +2.95)
- N=11 → N=39 (5年に拡張)、Sharpe は 7.57 → 2.95 へ
- 直近1年が異常に強かった (前回検証バイアス) が、長期では Sharpe +2.95 と継続レベル
- 平均 +186.7bps と1トレードあたりは依然強い

### ✅ bank_absorption: 期間拡張で堅牢性確認 (Sharpe +1.84 → +3.94)
- N=908 (5年×22銘柄) と最大規模
- 前回開発時の Sharpe +1.84 は控えめな推定だった
- WR 58.9%、平均+131.8bps、PF 2.06 と非常に安定

### ✅ orb_breakout_long, vwap_morning_meanrevert, eneos_vwap_trend
- いずれも Sharpe ≥ 2.0 を維持、長期検証でも堅牢
- 三井金属の Sharpe +1.96 は基準スレスレで要監視

## 解釈・示唆

1. **「直近データだけで判断する」検証は危険**
   - topix_overnight が典型例。1年では Sharpe +6.27 でも5年では +0.58
   - 戦略採用時は **必ず最低3年以上** で検証すべき
2. **JQuantsへの移行で大幅な乖離なし** (eneos/orb/vwap_meanrevert)
   - データソース変更で値が変わるリスクは限定的だった
3. **bank_absorption は期待以上**: 5年×22銘柄でも Sharpe +3.94 安定
4. **pair_portfolio は再構成必要**: 勝ちペアのみに絞れば Sharpe ≥ +5 が狙える
