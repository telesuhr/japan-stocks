## 🎯 結論と次ステップ

### 今回分かったこと
1. **半導体銘柄の寄付ブレイクアウトは反転する** (強い統計的証拠: t=-4.01)
2. 順張りブレイクアウトは**ほぼ確実に負ける** (5銘柄中4銘柄でマイナス)
3. LME-非鉄連動は**わずかな正のエッジ**があるが実務エッジ不足
4. SOXギャップフェードは銘柄選別が困難

### ⭐ 有望候補: ORB Fade 戦略

**仕様案**:
- 対象: 6503 三菱電機 (単独、Sharpe+1.94) or 半導体5銘柄バスケット
- シグナル: 9:00-9:30 の High/Low を 9:30以降にブレイク
- エントリー: ブレイク方向と**逆**に成行
- 決済: 15:30 引成 (or 反対方向ブレイクで損切)

### 次ステップ (ORB Fade 改良)

1. **サイズ感応度** — ブレイク幅による層別 (大幅ブレイクほど fade 効く?)
2. **時間帯分析** — 9:30ブレイク vs 10:00以降ブレイクで効果差
3. **早期決済** — 引けまでホールドでなく +30min or +1h で改善?
4. **市場環境フィルタ** — VIX/TOPIX方向でサブセット分析
5. **OoS検証** — H1/H2 分割で将来再現性確認

**Sharpe+1.01 → +2.0 に乗るパラメータ/フィルタが見つかれば採用候補**。
6503 単独なら既に +1.94 なので、条件最適化で突破可能性あり。

### ボツ戦略 (今回結論)
- ① SOX寄付フェード: エッジ弱く銘柄依存
- ② LMEアジア時間連動: サンプル大きいがエッジ小
