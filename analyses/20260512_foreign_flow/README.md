# A. 投資部門別売買データ × 翌週TOPIX戦略

## 結論を先に

✅ **複数の有望シグナルを発見** (勝率60-70%, Sharpe 1.4-2.2)。  
ただし t値は1.3-1.6 程度で**統計的に有意水準には届かず**。実戦投入には追加検証が必要。

## 解釈

### 1. **「海外買い・個人売り」のダブル条件が最強**

最も筋が良いのは **H4 (海外-個人 スプレッド)**:
- 勝率70.8% (Z>2.0)
- 「機関買い・個人損切り」の組み合わせ = 強気サイン
- このパターンが過去24週発生 → 17週で TOPIX 翌週上昇

### 2. **「投信買い越し Z>1.5」が高Sharpe**

- N=42、勝率64.3%、平均+79bps/週
- 投信フローは個人の代理 (積立NISA等)
- 投信に資金が入る局面 = リスクオン環境

### 3. **「自己売買 (Prop) 大幅売越し」もシグナル化可能**

- N=29、勝率55.2%、平均+73bps (ショート)
- 自己売買が積極的に売っている = HFT等の警戒シグナル

### 4. **海外勢買い越しは古典的に効く**

- H1z: 海外Bal Z>1.5 → ロング: Sharpe 1.41, 勝率 68%
- ただし**逆方向 (Z<-1.5 ショート) は機能しない** (Sharpe -0.95)
- 非対称: 「外人買い続伸」は予測可だが、「外人売りで反落」は予測しにくい

## 限界・注意点

1. **t値が低い**: 統計的有意性は十分とは言えない (t=1.3-1.6, p=0.1-0.2)
2. **公表遅延**: 金曜引け後公表 → 月曜開始 → 既に織り込まれている可能性
3. **長期データ依存**: 10年分での結果。直近の構造変化に弱い可能性
4. **方向の非対称性**: 買い方向はOK、売り方向は機能しない
5. **取引コスト**: TOPIX ETF (1306) で約3-5bps片道 → 往復6-10bps
6. **資金量**: 国内最大級のETFでも数十億円規模だと市場インパクトあり
