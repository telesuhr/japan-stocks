# マルチタイムフレーム (MTF) 戦略

## 結論を先に

✅ **22ルール検証で9つが有意プラス** (p<0.05かつnet>0)。  
✅ **Tier S+ ルール 2つ** (Sharpe 8.78 / 6.79) で実戦投入レベル。  
❌ **逆張り系 (Counter) はすべて失敗** (特にF1/B1は壊滅的)。

→ 結論: **MTFは「順張り (Confluence + Pullback)」で機能、逆張りは禁忌**。

## 検証方法

### 時間軸とその指標

| 時間軸 | 指標 |
|---|---|
| 週足 (5日) | week_ret = 過去5日の累積リターン |
| 月足 (20日) | month_ret, dist_ma20, rsi14 |
| 日足 (前日比) | daily_ret, gap_pct, prev_close比 |
| 時間足 (10:00, 13:00) | cum_ret_10:00, cum_ret_13:00 (寄付以降) |

### データ
- 14銘柄 (非鉄7 + AI半導体7)
- 2024-05〜2026-05 (約2年)
- 6,883 (date, code) 観測
- コスト: 4bps (片方向)

## 主要発見

### Tier S+ (Sharpe > 5) — 実戦投入候補

#### D2: 月下落 + 日反転 → Long (Sharpe 8.78)
```
条件:
  - 月足リターン (20日) < -5%
  - 日足リターン (前日比) > +2%
判定: 10:00 (前日close→当日寄付ギャップ判定)
保有: 10:00 → 引け
期待: +95bps/トレード, 勝率71%, N=348/2年
```
解釈: **「中期下落していた銘柄の日足反転」= 機関のリバージョン買い**。月レベルで売られすぎた銘柄が日足で反転した瞬間は強いシグナル。

#### A1: 週up + 日up + ギャップup → Long (Sharpe 6.79)
```
条件:
  - 週リターン > +2%
  - 前日比 > 0
  - 当日ギャップ > +0.5%
判定: 10:00
保有: 10:00 → 引け
期待: +73bps/トレード, 勝率68%, N=1280/2年
```
解釈: **完全 Confluence (整合)**。全時間軸が上昇で揃った時の続伸力は強力。

| A4: 週down + 日down + ギャップdown → Short | 4.62 | 下落も Confluence で続落 |
| A2: 週強up + 10:00 up → 引けまで Long | 3.56 | A1 から日足条件を緩めた版 |
| B2: 週up + 当日ギャップ↓ → 引けまで Long | 3.17 | 押し目買い (ギャップ埋め期待) |

### Tier A (Sharpe 2-3)

|---|---|
| A3: 週up + 日up + 13:00 up → 引けまで Long | 2.64 |
| D1: 週下落 + 日反転 + ギャップ↑ → Long | 2.38 |

## 失敗ルール (要禁忌)

### 大失敗 (Sharpe < -10)

| ルール | Sharpe | 教訓 |
|---|---|---|
| **B1: 週up + 前日下落 + 朝反発 → Long** | **-16.46** | 「朝反発」を信じて買うのは罠 |
| **F1: 週up + 月up + 日下落 + 朝反発 → Long** | **-13.79** | 上記同様、朝反発フィルタは逆効果 |

### Counter (逆張り) は全滅

| ルール | Sharpe |
|---|---|
| C1: 週過熱 + ギャップ↑ → Short | -5.08 |
| C2: 月過熱 + ギャップ↑ → Short | -2.15 |
| C3: RSI>70 + 朝強い → Short | -2.46 |
| C4: 週深売り + ギャップ↓ → Long (逆張) | -3.42 |
| F2: 週down+月down+日上昇+朝失速 → Short | -10.97 |

→ **過熱・反転狙いは何をやっても効かない**。  
日本株のMTFは「強い時はもっと強く、弱い時はもっと弱く」というモメンタム支配型。

## 解釈と教訓

### 1. 順張り Confluence が圧勝

- **すべての時間軸が同方向で揃った時**が最強シグナル
- 週up + 日up + ギャップupで続伸 (Sharpe 6.79)
- 週down + 日down + ギャップdownで続落 (Sharpe 4.62)
- 「方向が一致しているほど自信」という直感は数値的に正しい

### 2. Pullback の見極めが鍵

- 「週up + ギャップ↓」(B2, Sharpe 3.17) は機能 → ギャップ埋め
- 「週up + 前日下落 + 朝反発」(B1, Sharpe -16.46) は失敗 → 朝反発を待つのは遅すぎる

つまり**「ギャップで判定する押し目買い」は OK、「朝反発を確認してから入る押し目買い」は遅い**。

### 3. Trend Reversal (D2) が異常に強い

- 月下落 + 日反転 (Sharpe 8.78) = 22ルール中最強
- これは **機関の押し目買い** を捉えている可能性
- 月レベルで売られすぎた銘柄に対する反転シグナルは強力

### 4. 逆張り系の失敗の意味

- 過熱状態 (RSI>70, 週ret>5%) でも続伸する
- ギャップアップへの逆張りは破滅的
- → 日本株では「**強い動きには素直に乗る**」が原則

## 推奨統合戦略

### MTF-Combined: 順張り3戦略合成

```
毎日10:00時点で以下を判定:

[Long エントリー条件 (OR)]
  L1: 月足-5%以下 + 当日日足+2%以上 (D2)
  L2: 週足+2%以上 + 前日比+0% + ギャップ+0.5%以上 (A1)
  L3: 週足+2%以上 + ギャップ-0.5%以下 (B2押し目)

[Short エントリー条件]
  S1: 週足-2%以下 + 前日比<0 + ギャップ-0.5%以下 (A4)

保有: 10:00 → 引け
コスト: 4bps
期待: Sharpe 5+ (合成効果)
取引頻度: 年100-200回程度
```

### 各ルールの単独運用も可能

最も安定: **A1 (Sharpe 6.79, N=1280, 勝率68%)** ← 取引機会も多い  
最高リターン: **D2 (Sharpe 8.78, N=348)** ← 機会少ないが当たれば大きい

## 限界・注意点

1. **OOS未検証**: 2年分の同一データで全数検証 → 過学習可能性
2. **N が大きい**: A1 (N=1280) などは統計的に強いがレジーム依存可能性
3. **コスト4bps**: 大型流動性銘柄前提。中小型では現実コスト6-10bps
4. **大引け持ち越し**: 引け成行スリッページ未反映
5. **時間軸の閾値感度未検証**: 週リターン±2%は最適か?
6. **銘柄バイアス**: 14銘柄 (非鉄+AI半導体)中心 → 他セクターは未検証
