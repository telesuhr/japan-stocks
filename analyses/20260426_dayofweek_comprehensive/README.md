# 全セクター × 曜日別 × ON / イントラ / 全日 包括的分析

**分析日**: 2026-04-26
**データ**: NAS MariaDB `refinitiv_news.intraday_data` (2025-04-01〜2026-04-24)
**対象**: 6セクター 27銘柄 — CORE5 / 非鉄 / 半導体 / エネルギー / 海運 / 内需
**手法**: 等加重バスケット日次 ON・イントラ・全日リターン × 曜日別 t検定

---

## 目的・仮説

既存の曜日分析 (20260422_dayofweek_*) はLMEシグナル条件付きや個別セクター限定だったが、
本分析では **無条件の曜日効果** を **全6セクター・3リターン種別で網羅的に検証** する。

- ON (オーバーナイト): 前日引け → 当日寄り
- イントラデイ: 当日寄り → 当日引け
- 全日: 前日引け → 当日引け (ON + Intra の合成)

### B. 木曜ON が無条件では強い — LMEシグナル条件付きとの逆転 (★★ 重要発見)

| セクター | 木曜ON bps | t |
|---|---:|---:|
| CORE5 | **+52.4** | **+4.61** |
| 非鉄 | +61.3 | +3.02 |
| エネルギー | +41.8 | +3.09 |
| 半導体 | +40.5 | +2.64 |

**無条件の木曜ONは全セクターでプラス・有意。**
以前の分析 (20260422_dayofweek_lme) での「木曜 Sharpe-1.31」は **LME銅+1%という条件付き** の結果。

> **解釈**: LME銅が大きく上昇した翌日 (木曜) は日本株がすでに前日に織り込んでしまい、
> 寄付後の追加上昇が出にくい (イントラデイが弱い)。しかし **無条件の木曜ONは強い**。
> 木曜のLMEデータ (水曜LME終値) が翌朝の寄付を押し上げている可能性が高い。

→ **修正ルール**: 「木曜除外」はLMEシグナル戦略のイントラデイ戦略に限定。
　木曜のONロング (夕方〜翌朝) は別途有効である可能性がある。

## 解釈・示唆

### ONとイントラの性格差
- **ON強い曜日 (水・木)**: LME・海外市場の情報が夜間に入り、寄付で織り込まれる
- **Intra逆行する曜日 (火)**: ONで上げた分を日中に吐き出す (MON EFFECT)
- **Intraが独立してプラスの曜日 (水非鉄・金エネ)**: 日本時間の取引時間中に買いが集まる構造

### セクター別特性
| セクター | 最強曜日 | 弱い曜日 | コメント |
|---|---|---|---|
| 非鉄 | 水 (全日+113bps) | — | LME水曜価格が木曜早朝に確定→木曜ON強 |
| 半導体 | 水 (全日+80bps) | — | SOX連動、週半ばに方向性が出やすい |
| エネルギー | 木 (全日+68bps) | — | 原油週次在庫 (水曜夜発表) の翌日効果 |
| 海運 | 水ON (+27bps) | 月Intra (-50bps) | 週明け売り vs 週中買い |
| CORE5 | 木ON (+52bps) | 火Intra (-34bps) | 景気敏感全般の曜日パターンを集約 |
| 内需 | — | 火Intra (-34bps) | 景気敏感LSのShort leg として木・水に活用 |

## 限界・注意点

1. **無条件曜日効果**: LMEや先物方向と組み合わせると効果が変わる (既存分析参照)
2. **木曜ONの解釈**: LME銅+1%条件付きでは木曜は除外推奨のまま。無条件ONのみプラス
3. **N問題**: 1セクター×1曜日のNは約120-250。t統計量とセットで判断を
4. **期間**: 2025-04〜2026-04 の1年分。2026-02 イラン戦争後レジームを含む
5. **コスト未考慮**: 表示値はコスト前。実運用では -4bps (片道2bps×往復)
