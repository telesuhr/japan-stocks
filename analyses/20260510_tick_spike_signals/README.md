# A. ティックスパイクシグナル検証

## 目的・仮説

ティックデータが新たに使えるようになったため、**1分間の約定回数(tick_count)** を新しい特徴量として加え、5分足では捉えきれなかったエッジが存在するかを検証する。

仮説:
- H1) 約定回数スパイク × 上昇 → 買い継続
- H2) 約定回数スパイク × 下落 → 売り継続
- H3) スパイク → 反転 (パニック売り後リバーサル)

---

## データ概要

- 期間: 2026-01-05 〜 2026-04-30 (4か月)
- 銘柄: トヨタ・ソフトバンクG・MUFG・アドバンテスト・ソニーG (流動性主要5銘柄)
- 元データ: JQuants ティック (`~/Data/jquants_trades/`) から DuckDB 直接集計
- 1分バー: 118,231 (取引時間内のみ)
- 特徴量付き: 114,281 レコード
- コスト: 往復4bps

### 追加特徴量

| 特徴量 | 内容 |
|---|---|
| `tick_count` | 当該1分内の約定回数 |
| `tick_ratio` | tick_count / 直近30分の平均 |
| `vol_ratio` | volume / 直近30分の平均 |
| `bar_ret` | 当該1分のリターン (close/open) bps |

## 核心的発見: 上下非対称性

**下方向**のティックスパイクは継続する (モメンタム)、**上方向**は予測力ゼロ。

### 下方向 (パニック売り) → モメンタム継続

```
A2 (3xスパイク × 下落):
  fwd_5min:  +0.75bps net (t=3.13) ✓
  fwd_15min: +2.76bps net (t=2.27) ✓
  fwd_30min: +6.55bps net (t=2.88) ✓ ← 30分保有で十分なエッジ

A4 (5xスパイク × 下落):
  fwd_5min:  +4.96bps net (t=2.56) ✓
  fwd_15min: +19.43bps net (t=3.16) ✓ ← 強いシグナルほど大きく
```

→ **約定回数スパイク + 下落 = パニック売りの始まり。15-30分は継続する。**

### 上方向 (FOMO買い) → 予測力なし

```
A1 (3xスパイク × 上昇):  すべて t<1 で非有意
A3 (5xスパイク × 上昇):  むしろ負け方向 (-10bps)
A7 (ティック+出来高 × 上昇): すべて t<1
```

→ 上昇時のスパイクは「FOMO買いがすぐ反転される」可能性を示唆するが、リバーサル方向(A5)も非有意。**上昇スパイクは予測力ゼロ**と結論。

### 大口主導 vs 小口殺到

```
A11 (出来高High × ティックLow × 上昇 = 大口主導): 非有意 (-15bps)
A9  (ティックHigh × 出来高Low × 上昇 = 小口殺到): 非有意
```

→ 出来高とティック数の組み合わせでは、上昇方向は判別できない。

## なぜ前回(5分足)で見えなかったか

前回の分析 (`20260509_intraday_rules`) では「5分足のvol_ratio」を特徴量に使っていた。今回:

1. **1分粒度に細かく**: 5分→1分でシグナル感度向上
2. **出来高ではなく約定回数**: 出来高は大口1発で歪むが、tick_count はより安定した「市場の興奮度」を測る
3. **下方向限定**: 上昇時のシグナルを除外することで信号対雑音比改善

## 実装イメージ (最有力: A2)

```
条件 (1分バー終了時にチェック):
  当該バーの close < open (下落)
  AND 当該バーの tick_count >= 直近30分平均 × 3

エントリー: 当該バー終値で空売り
エグジット: 30分後の close で buy back
期待: +6.55bps/トレード, Sharpe=10.4
N: 銘柄あたり約88件/4か月 = 月22回程度
```

---

## 限界・注意点

1. **5銘柄のみ**: 流動性主要銘柄に偏っている。中小型株では結果が異なる可能性
2. **N=127 の問題**: A4 (5xスパイク) は強いが標本が少ない。将来再現性は要確認
3. **上下非対称**: 上昇方向のエッジが見えないのは「下落相場が続いた期間」というサンプル偏りの可能性
4. **30分保有のリスク**: 場中ニュースで急反転するリスクあり (損切り設定が必要)
5. **執行コスト**: 4bpsはconservative。空売りには貸株料も別途必要 (年率1〜3% = 0.3〜0.8bps/日)
