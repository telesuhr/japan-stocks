## 核心的な意味

### 戦略は「方向の固定」が間違っている

これまでの分析は全て **「大口プリント = ロング」** という前提だった。  
だが今回判明したのは:

```
上昇/横ばい相場 → 大口買いプリント → ロング が機能 (1-4月)
下落トレンド相場 → 大口売りプリント → ショート が機能 (5/11)
```

つまり「**シグナル定義は正しいが、方向の固定が間違い**」。

### 「大口売り」の特殊性

これまでの分析 (`20260510_large_prints`) で:
- 「大口売り → 30分後反転買い」が機能 (学習期で +0.01bps net, t=10.85)

しかし 5/11では:
- 「大口売り → 30分後 **継続売り**」が **+228bps, 100%勝率**

これは**反対**の結果。なぜか?

→ **下落トレンドが既に確立していれば、大口売りは反転せず継続する**

学習期 (1-4月) は概ね横ばい〜上昇基調 → 大口売りは「**カルテルの売り**→反発」  
5/11 のような暴落日 → 大口売りは「**真のパニック売り**→継続」

## 必要な戦略改造

### Phase 1: レジーム判定モジュール

朝の地合いを判定するロジック:

```python
def detect_regime(bars, t):
    """その時点までのバーから当日の地合いを判定"""
    today_bars = bars[bars.date == t.date()]
    cum_ret = (today_bars.close.iloc[-1] / today_bars.open.iloc[0] - 1) * 10000

    # TOPIX も加味すべきだが、まずは銘柄単独で
    if cum_ret > +50:
        return 'UP'        # 上昇トレンド
    elif cum_ret < -100:
        return 'DOWN'      # 下落トレンド
    elif cum_ret < -50:
        return 'WEAK'      # 弱含み
    else:
        return 'NEUTRAL'   # 様子見
```

### Phase 2: レジーム別シグナル切替

```
UP/NEUTRAL: 大口買い → 30-60分ロング (B-Optimized, D2, D3)
DOWN:       大口売り → 30分ショート (今回の発見)
WEAK:       エントリー停止 (5/11 9:00-9:05 のような「判定不能ゾーン」)
```

### Phase 3: 寄付直後の判定保留

5/11 で B-Improved が止めきれなかった「9:00直後のシグナル」問題:

```
09:00 - 09:05: 完全停止 (まだ地合い判定不能)
09:05 - 09:15: cum_ret_5min が出れば判定可
09:15 以降: レジーム別運用
```

## 「大口売り 100%勝率」を疑う

50戦全勝は**異常**。原因は:

5/11は終日強い下落トレンド → 「下落バー × 大口あり」のすべてが下落継続を予言してた  
→ **戦略ではなく単なるトレンドフォロー** (下げが続けばどこでもショート勝ち)

つまり 100%勝率は「**この日の特性**」であり、別日では再現しない可能性が高い。  
ただし方向感は明確: **下落トレンドではロング戦略を停止し、ショート側に乗ること**。
