# アルゴ指紋プロファイリング 総合レポート

**実施日**: 2026-04-23
**期間**: 2024-11 〜 2026-04 (約 17 ヶ月、1 分足 OHLCV)
**対象**: 最大 20 銘柄 (半導体/非鉄/通信/自動車/商社/銀行) + TOPIX 指数

---

## TL;DR

1. **1 分 OHLCV だけで「アルゴのタイプ」は判別可能** — ただし「どの業者か」は不可
2. **k=4 クラスタで日本株大型は綺麗に分類された**:
   - 流動性薄・インパクト大 (中型半導体/非鉄)
   - **指数連動・MOC 支配 (大型優良: トヨタ/SMFG/商社)**
   - TWAP/VWAP 執行系 (中堅)
   - **HFT 支配・超流動性 (ソフトバンク 9434 のみ完全外れ値)**
3. **引け前は唯一モメンタム化する時間帯** — 他時間帯は全て微 MR
4. **semi_disp 動的アロケーションの発見は look-ahead バイアスだった** — 同日 dispersion と戦略 PnL の相関は予測シグナルではない
5. **コスト無視の意思決定用では 15-30 分足がブレイクの騙し回避に優秀** (追従率 60% vs 1-5 分足 50%)

### 時間帯別 解釈

- **寄付 (S1)**: 個別材料動き (指数連動最弱 0.36)、大口持続 0.50 = 方向が出る
- **前場 (S2)**: 出来高ピーク 30%、**だましブレイク 62%** (最高)
- **後場 (S3)**: 指数連動ピーク 0.47 = ETF/パッシブ裁定の時間帯
- **引け前 (S4)**: **唯一モメンタム** (ρ1 +0.006)、流動性最深 (Kyle λ 最小)

**実戦帰結**:
- ブレイク狙いは **S3 後場 or S4 引け前** (だまし率最低)
- セクター個別狙いは **S1 寄付** (指数連動が弱く alpha が残る)
- 大口執行は **S4 引け前** (市場影響最小)

### ⚠️ レジーム分析の落とし穴

D で発見した「semi_disp HI 日に D_freq が Sh +10 / LO 日 +0.2」の大差は、**同日知識**を使った分析だった。
E で前日値によるバックテストに修正したところ:
- Dynamic Sh **5.82** vs Static Sh **6.23** で**悪化**
- D_freq の HI/LO が逆転 (+8.34 → +2.77)

**教訓**: intraday dispersion は co-incident indicator であって predictor ではない。
リアル運用では前日の dispersion や複数日 MA など、予測性のある変換が必要。

### 意味のある相関

lasertec_ma25 の -0.38 (topix_day_ret) は MA25 戦略が下げ相場の逆張り trade で儲けた結果の co-movement で、真の予測力ではない可能性。時系列的に厳密に検証する必要あり (次課題)。

## 実運用への提言

### 即座に使える知見

1. **既存戦略のエントリー時間を最適化**
   - ブレイク系戦略は **13:00-14:30** にフィルタ (だまし最少)
   - 寄付ペア系は **09:00-09:15** 集中実行 (指数連動最弱)

2. **銘柄選定をクラスタベースで**
   - ペア LS → Cluster 0 (流動性薄) 銘柄から選ぶ
   - VWAP MR → Cluster 2 (TWAP 執行系) 銘柄で試す
   - Cluster 1 (大型指数連動) は α 取得困難と認識
   - ソフトバンク 9434 は **1 分足裁量取引対象から除外**

3. **引け前 30 分の特性を意識**
   - 唯一モメンタム化する時間帯 → ブレイク追随エントリー可
   - 流動性最深 → 大口執行はここで (実運用時のスリッページ最少)

### 注意すべき非発見

- **1 分 OHLCV では HFT の直接検出は困難** (ρ1 全て ≈ 0)
- **板/ティックデータが必要な分析トピック多数** (アイスバーグ、スニッピング等)
- **同日 dispersion は予測シグナルではない** (look-ahead 注意)

### 次の研究候補

1. **実データクリーニング強化** — 5803 3/3 のような異常値の自動検出・除外
2. **VWAP 引寄指標の再設計** — 短期 rolling を引いた residual の半減期
3. **SQ 日・月末リバランス日のフラグ化** — これらの日だけの指紋プロファイル比較
4. **tick data 取得検討** — アイスバーグ検出・真の HFT 活動度
5. **lasertec_ma25 の -0.38 相関の因果構造検証** — 予測力 vs 同時反応
