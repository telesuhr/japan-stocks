# 出来高 → 方向性 予測力の検証

**対象**: 非鉄3 + 半導体5 = 8銘柄 (5711/5706/5713/8035/6857/6146/4063/6963)
**期間**: 2025-04-01 〜 2026-04-21 (約248営業日)
**評価**: Spearman相関 + 5分位バケット + t-stat

---

## 🎯 結論: **出来高単独では方向予測できない。ただし「ギャップ×出来高」の組合せは強いエッジを持つ**

| シグナル | 集約 Spearman | 判定 |
|---|---:|---|
| S1. 寄付30分vol_ratio → 残り時間 | **-0.015** (p=0.51) | ❌ 実質無相関 |
| S2. 1分足スパイク (3x+) → 直後 | ≈ 0 (N≈19k) | ❌ 単独では方向示唆なし |
| S3. 寄付加速度 (9:15-30/9:00-15) → 残り時間 | **+0.010** (p=0.65) | ❌ 実質無相関 |
| S4. 前場vol_ratio → 後場 | **+0.012** (p=0.60) | ❌ 実質無相関 |
| S5. ギャップ × 出来高 4象限 | t=+2.77〜+3.05 | ✅ **強いエッジあり** |

**核心的発見**: 出来高は「**方向性の種類**」を決めるのではなく、「**ギャップの真剣度**」を測るメタ情報として機能する。

## 1. シグナル別サマリー

### S1. 寄付30分の出来高レシオ → 残り時間方向

集約 Spearman r = **-0.015** (p=0.51, N=1963) → 実質無相関

5分位バケット別 fwd_bps (9:30→15:30):
| bucket | N | mean_bps | WR | t |
|---|---:|---:|---:|---:|
| Q1(低) | 393 | +20.6 | 58.5% | +2.78 |
| Q2 | 392 | -2.1 | 48.0% | -0.26 |
| Q3 | 393 | +15.3 | 52.7% | +1.60 |
| Q4 | 392 | +26.4 | 56.6% | **+2.81** |
| Q5(高) | 393 | -4.8 | 49.4% | -0.48 |

- **非単調**: Q4 (1.3〜2倍程度) が最良 +26.4 bps、Q5 (極端に高い) では消える
- 銘柄別で有意 (p<0.05) なのは **三菱マテリアル (5711) r=-0.137** のみ
- 解釈: 「異常な出来高」は情報不確実性の印で、方向性はむしろ失われる

### S2. 1分足出来高スパイク (20bar中央値の3倍以上) → 直後5/30分

集約 (N≈19,000):
- **up-spike → 5min mean = -0.3 bps (t=-1.52)** → 買い圧スパイクは直後やや反転
- **down-spike → 5min mean = ≈0 bps (t=+0.10)** → 売り圧スパイクは方向示唆なし

スパイク倍率 2x / 3x / 5x / 8x / 12x と上げても、集約レベルでは mean 改善なし → スパイク強度と方向性は独立。

**個別銘柄で目立つもの** (t|≧2 のみ):
| 銘柄 | spike方向 | horizon | mean | t | パターン |
|---|---|:-:|---:|---:|---|
| 6857 アドバンテスト | up | 5min | -1.81 | **-2.90** | 強烈なリバーサル銘柄 |
| 6857 アドバンテスト | down | 5min | -1.48 | **-2.03** | 下落スパイクは継続 |
| 6146 ディスコ | up | 5min | -1.14 | **-2.03** | 上昇スパイクは失速 |
| 5711 三菱M | up | 5min | -0.63 | **-2.11** | 弱いリバーサル |
| 6963 ローム | down | 5min | +1.59 | **+2.00** | 下落スパイクは戻す |

→ スパイクの方向効果は **完全に銘柄特性** (リバーサル型/モメンタム型) に依存。
6857/6146/5711 はアップスパイク反転型、6963 はダウンスパイク反転型。

### S3. 寄付加速度 (9:15-9:30 vol / 9:00-9:15 vol)

5分位バケット:

| Q1(低) | 394 | +8.3 | +1.18 |
| Q2 | 394 | +3.7 | +0.45 |

- Q5 の極端に加速した日は効果消滅 (むしろマイナス) — S1と同じパターン

| Q1(低) | 396 | +6.6 | +1.42 |

### S5. ギャップ × 出来高 4象限 → 寄付〜大引け方向 ⭐

| UG_LV | UpGap + LowVol | 544 | +0.5 | 48.0% | +0.05 |

- **UG_HV**: 上ギャップ+高出来高 → **継続** (+40 bps, t=2.8)

## 2. 実行可能な戦略候補 (大ギャップ日のみ)

寄付直後30分の出来高レシオを見て、4パターンに振り分け:

| 条件 | 方向 | 期待mean_bps | コスト4bps後 |
|---|:-:|---:|---:|
| 大UpGap + 低出来高 (Q1/Q2) | **Short** | +44.6 (フェード) | +40.6 |
| 大UpGap + 高出来高 (Q4/Q5) | **Long** | +26.9 | +22.9 |
| 大DownGap + 高出来高 (Q5) | **Long** | +65.4 | +61.4 |
| 大DownGap 全般 (Q2以上) | Long | +47.4 | +43.4 |

**機会頻度**: 大ギャップ日 ≈ 上下位20% = 年間約50日/銘柄。8銘柄なら年400トレード。
**Sharpe目安**: mean +45 bps, std ≈ 150 bps で Sharpe ≈ 0.3/day × √252 → **年Sharpe ≈ 4〜5** の可能性あり (粗い見積もり)。

## 3. なぜ出来高単独では方向予測できないのか

1. **出来高はボラティリティの指標であって方向性の指標ではない** — どちらの方向にも大きく動きうる
2. **高出来高=機関の両サイド取引**が入ることが多い — ブレイク後に戻されやすい
3. **イベントドリブン** (決算、経済指標) は方向性が事前に分からない
4. **高出来高日の非線形U字パターン**: 中庸な出来高が最良、極端な出来高は効果消滅 → 情報不確実性が上昇

## 4. なぜ「ギャップ×出来高」は機能するのか

- **UpGap + HighVol**: 本物の材料 + 機関の参加 → 継続
- **UpGap + LowVol**: 薄商いの空中戦、材料薄 → フェード
- **DownGap**: 日本株固有のオーバーナイト売られ過ぎ傾向 + 国内買い戻し → 全般的に戻す
- **DownGap + HighVol**: パニック売りの過剰反応 + 強力な買い戻し → +65 bps戻し

→ **出来高は価格情報 (ギャップ) の "確度スコア" として使うのが正解**

## 6. 注意事項・限界

- N=56/象限 は統計的に限界的。特に大UpGap×LowVol の-52 bps は過学習リスク
- イラン戦争期間 (2026-02-28〜) を含む — 期間除外での再検証推奨
- ギャップ閾値 106/-78 bps は8銘柄集約の分位点。銘柄別では異なる
- vol_ratio の20日中央値ベースラインはFOMC等のセクター共通高出来高日に影響される
- **コスト4bpsで評価済み、現実のスリッページ次第ではエッジ縮小**
