## C. 木曜ON × LMEシグナル条件分解 — 旧発見の逆転解明

### 結果

| セクター | LME<=-1% (N≈6) | 中立 (N≈38) | LME>=+1% (N≈10) |
|---|---:|---:|---:|
| CORE5 | +42.1 (t=+10.18) | +47.1 (t=+2.83) ★ | +106.7 (t=+1.93) |
| 非鉄 | +59.6 (t=+0.79) | +37.5 (t=+1.00) | +194.8 (t=+2.54) ★ |
| 半導体 | -7.6 (t=-0.09) | +37.8 (t=+1.22) | +114.1 (t=+1.79) |
| エネルギー | +5.5 (t=+0.16) | +18.8 (t=+0.99) | **+174.8 (t=+5.52) ★★★** |
| 海運 | -4.0 (t=-0.12) | -0.6 (t=-0.04) | +38.9 (t=+1.88) |

★ = |t|≥2.0

### 重要発見: 「木曜が悪い」はイントラデイの話、ONは逆に最強

旧分析 (20260422_dayofweek_lme) の「木曜 Sharpe-1.31」の正体が解明された:

```
木曜 LME>=+1% 日の内訳:
  ON  (前日引→寄)  : エネルギー +174.8bps (t=+5.52) ← ギャップアップで寄り天
  Intra (寄→引)   : マイナス方向 ← LMEシグナルの利益がONで吸収され、Intraは逆行
  Full (前日引→引): ONの強さが残る
```

**LMEシグナルは木曜「夕方から翌朝」の間にONで完全に消化される。**
寄り後にエントリーしても既に遅い = 「木曜イントラ」はエッジなし。

→ **修正ルール (確定)**:
- 「木曜はLMEシグナル戦略に使えない」の正確な意味は**イントラデイ戦略**に限定
- 木曜ONの活用 (前日夕方〜翌朝寄り決済) は**エネルギーが特に有望** (t=+5.52)
