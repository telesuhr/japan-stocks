## OR戦略（Overlay）の設計案

```
毎朝 09:05 時点:
  ① TOPIX夜間ギャップ ≥ +0.3%   → topix_overnight 実行
  ② LME累積10日 ≥ +3%           → lme_momentum 実行（①と重複しない場合）

毎日 15:25 時点:
  ③ LME東京時間変化率 ≥ +1.0%   → lme_on_copper 実行（優先、①②をキャンセル）

重複処理:
  - ③が発動 → ①②をキャンセル（lme_on_copper優先）
  - ①と②が同日発動 → いずれか一方のみ（lme_on_copperと同様にダブルベット禁止）
  - ポジションサイズ: 常に¥5,000万固定（重複でも増やさない）
```

### OR戦略の期待値

| 指標 | lme_on_copper単独 | OR戦略 |
|------|:-----------------:|:------:|
| 月間発動回数 | 月1〜2回 | 月3〜4回 |
| 1トレードMean | +137bps | +62bps |
| 年間期待P&L | ¥150〜250万 | ¥270〜450万 |
| Sharpe | +12.34 | +7.14 |
| t-stat | +2.69 | +2.74 |

発動頻度が増える分、1トレードの質は下がるがt-statは維持される。
