# 日本株 曜日別特性: 2026年2月末イラン戦争 前後レジーム比較

## 背景
2026年2月末に勃発したイラン戦争を境にしたレジーム転換を検証。
前回分析 (20260422_dayofweek_regime) は 2025-06 の12日戦争基準だったが、
本分析は**より直近の2026年2月末開戦ショック**を対象とする。

## 期間定義
- **Pre** : 2025-04-17 〜 2026-02-27 (N=211日)
- **War** : 2026-02-28 〜 2026-03-13 (N=10日)
- **Post**: 2026-03-16 〜 2026-04-20 (**N=25日**, 標本極小)

⚠ **Post期間は曜日あたり4-6サンプルしかないため、統計的結論は慎重に。**

## 重要な発見: 前回チートシートの曜日構造は"崩壊"

前回 (2025-06戦争基準) の Post期間で見つかった:
- 「**水曜Long 本命**」(非鉄+113bps, t=+4.6)
- 「**火曜 寄り天Short**」(コア5 -48bps, t=-3.3)
- 「**月曜 海運Short**」(-51bps, t=-3.5)

これらが今回のイラン戦争後にどう変化したか:

| 旧本命シグナル | 2026年2月末戦争前 | 2026年戦争後 | 変化 |
|---------------|-------------------|--------------|------|
| 水曜 非鉄Long | +76bps | **-123bps** (N=5) | **逆転** |
| 火曜 コア5Short | +27.5bps上昇 → 寄り天 | **+58bps上昇 → 寄り天継続** | **継続** |
| 月曜 海運Short | -36bps | **-202bps (t=-3.72)** | **増強** |

### 新レジームの特徴
1. **月曜の全面安が極端化**: 海運 -202bps、コア5 -67bps、エネルギー -75bps
2. **木曜も弱化**: 旧レジームでは強日だったコア5木曜が -117bps に反転
3. **水曜は混在**: TOPIX全日は +275bpsだがN=5、セクターごとにバラバラ
4. **火曜の寄り天構造は継続**: 半導体-107bps (t=-2.15)

## 実運用への含意

### 機能している可能性が高い戦略 (継続推奨)
- ★ **月曜 海運Short (前場)**: 旧戦略でt=-3.5 → 戦後さらに強化 t=-3.72
- ★ **火曜 半導体Short** (t=-2.15): 寄り天構造継続、むしろ強まった
- ★ **月曜 コア5Short** (t=-2.63): 週初リスクオフの主要標的

### 機能しなくなった可能性が高い戦略 (停止検討)
- ✗ **水曜 非鉄Long**: +113bps → -123bps (逆転の可能性、要経過観察)
- ✗ **水曜 エネルギーLong**: +59bps → +4bps (消失)
- ✗ **木曜 景気敏感Long**: コア5が +26 → -117bps

### 新規に浮上した戦略候補
- 🆕 **木曜 景気敏感Short**: コア5 -117bps (t=-2.51) が新規出現
- 🆕 **月曜 エネルギーShort**: -75bps、統計非有意だがWar期間の示唆と整合

### LME銅/日経先物本命戦略への影響
本命の「LME銅+1% → コア5Long」(旧Sharpe+11.5) は:
- 月曜実行は-67bps/日のドラッグがかかる → **月曜除外フィルタ追加推奨**
- 木曜実行も-117bps/日のドラッグ → **木曜除外フィルタ追加推奨**
- 実質使える曜日は Tue/Wed/Fri のみになった可能性

## Post期間の標本薄問題

Post N=25日、曜日あたり4-6サンプル:
- 非鉄水曜 N=5 → -123bpsは1-2日の大きな動きに引きずられた可能性
- 海運月曜 N=17 (全銘柄×6日×3銘柄) → 比較的信頼できる
- 統計有意(|t|>=2)でも**今後1ヶ月の追加データで再検証必須**

## 実務フロー案 (保守的)

```
毎朝の判断:
  月曜 → 海運Short前場、コア5は寄り新規Long禁止、指数Longノーポジ
  火曜 → 半導体/コア5 寄り天Short候補、Long持越しは寄り利確
  水曜 → 判断保留 (旧本命消滅、N不足でエッジ不明)
  木曜 → 景気敏感Long禁止 (-117bps)、むしろShort検討
  金曜 → 中立、既存戦略通常運用

LME/マクロシグナル発動時:
  本命「LME+1%→Long」は月曜・木曜実行を除外
  火曜実行なら寄り利確、引け持越し回避
```
