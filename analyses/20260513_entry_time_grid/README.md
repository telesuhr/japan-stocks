## 重要な洞察

### 1. AI半導体は「朝モメンタム/午後収束」の二相性

- **朝 (9:40-10:00)**: Momentum (best→long, worst→short) が機能
  - 寄付ギャップ + 海外連動の方向が午後まで続く
- **昼 (10:30-13:00)**: 異なるビジネスモデル間のペア収束が機能
  - 同じ「半導体」でも装置/設計/受動部品で挙動異なる
  - 一時的な乖離は引けに収束

### 2. 非鉄は「寄付直後Reversion」

- **9:10 Reversion が Sharpe 0.78 で最強**
- LME銅夜間取引の動きが寄付ギャップに織り込まれる
- ギャップで上振れた銘柄は引けまでに修正される

### 3. ペアトレードの最適時刻は「ペアの性質次第」

| 関係 | 最適時刻 |
|---|---|
| 同じ業態 (装置メーカー同士) | 朝早い (9:20-9:50) |
| 異なる業態 (装置 vs 設計, 製錬 vs 電線) | 10:00以降 |
| 大幅に異なる (ソシオvs東エレ) | 午後 (12:50, 13:00) |

ペアが「似ているほど早く収束、異なるほど遅く収束」する。

## 限界・注意点

1. **In-sample 探索**: 2024-2026を学習 + 評価 → OOSで継続するか別途検証必要
2. **Sharpe 1.6 でも個別株分散リスク**: 単一ペアでは外れ年もある
3. **流動性**: 12:50判定の場合、ランチタイム明けの板の薄さに注意
4. **判定時刻精密化リスク**: 5/10分単位でのチューニングはオーバーフィット可能性
5. **大引け持ち越し**: 引け成行スリッページが現実コストに影響
