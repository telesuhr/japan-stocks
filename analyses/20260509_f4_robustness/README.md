# F4戦略ロバストネス検証

## 分析の目的

前分析で見つかった「F4: 前日-1%以下 × ギャップ-0.2%以下 → 引け買い翌朝売り (ON)」が、  
**関税ショック(2025年4月)に依存した特殊な結果なのか、通常期でも有効な戦略なのか**を検証する。

---

## データ概要

- 期間: 2025-01-01 〜 2026-05-07
- 銘柄: 日本株117銘柄
- F4条件合致件数: **3,197件**
- 関税ショック除外定義: 2025-04-02 〜 2025-04-25

## 総合評価

### 「勝てるか」という問いに対する答え

**条件付きで YES。**

1. **エッジは実在する**: ショック除外でもSharpe=1.55、t=6.21。パラメータ全域でt>5.7。
2. **ただし相場環境依存**: ショック前(2025年1~3月)は有意でなかった。**高ボラ・下落トレンドの局面で特に強い**。
3. **現在の環境(2025年4月以降)**は依然としてショック後の高ボラ環境が続いており、N=2481で Sharpe=1.65。**今は使えるタイミング**。
4. **最大リスク**: 2026年3月のような静穏な横ばい相場で -47bps/月 のドローダウン発生。

### 実装時の推奨設定

| 項目 | 推奨値 | 根拠 |
|---|---|---|
| 前日リターン閾値 | ≤-100bps (1%下落) | パラメータ感度で十分 |
| ギャップ閾値 | ≤-20bps | 適度な絞り込み |
| 対象セクター | 非鉄・化学・商社・IT・自動車 | ショック除外でSharpe>2.7 |
| 除外セクター | 鉄道・サービス・通信 | マイナスSharpe |
| エントリー | 15:00 引け成行 or 後場ザラ場TWAP | 終盤スリッページ注意 |
| エグジット | 翌日 9:00-9:05 寄付成行 | 開場直後の流動性確保 |
| ポジションサイズ | 複数銘柄分散 (1銘柄あたり資産の5%以内) | std=230bps の変動リスク |

## 限界・注意点

1. **ショック前の非有意性**: 静穏相場ではエッジが消える可能性
2. **月次レベルのドローダウン**: 2026年3月 -47bps、連続負け月のリスクあり
3. **セクター効果のN不足**: 化学N=14、鉄道N=13は統計的に不安定
4. **スリッページ未考慮**: 4bps コスト想定だが大口では引け成行で市場インパクトが出る
5. **制度変更リスク**: 2025年4月以降の相場構造変化が持続するかは不明
