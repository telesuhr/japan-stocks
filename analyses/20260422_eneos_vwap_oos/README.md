## ⚠️ バグ修正履歴

**初回 run.py に将来データリークのバグがあった。修正後の数値が正。**

| 指標 | バグあり(初回) | **修正後(正)** |
|------|:---:|:---:|
| N | 90 | **69** |
| Mean | +144bps | **+38.6bps** |
| WR | 86.7% | **58.0%** |
| Sharpe | +17.13 | **+5.54** |
| t-stat | +10.24 | **+2.90** |

**バグ内容**: `sig_bar = morning[morning.index.hour <= signal_hour]` の条件が  
`hour == 9` なら 9:00〜**9:59全バー**を取得し `iloc[-1]`（≈9:59）でシグナル判定していた。  
正しくは `(hour == 9) & (minute <= 30)` で **9:30以前のバー**のみに限定する必要がある。  
エントリーバーの取得条件も同様に修正。
