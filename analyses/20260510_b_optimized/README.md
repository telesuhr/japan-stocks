## 目的

深掘り5項目分析の知見を全部積み上げて、B戦略の最終形を作る。  
旧B1 (Sharpe=3.14, +2.34bps net) からどこまで磨けるか。

### 想定外の発見: **tick_ratio フィルタは不要**

深掘り分析で「tick_ratio≥1.5 で絞ると悪化 (-2.47bps)」が出ていたので、逆方向に「tick_ratio<1.5 で絞れば改善」と予想していたが、**実際は両方向のフィルタとも全体パフォーマンスを下げる**。

つまり tick_ratio という変数自体が**B戦略では予測力を持たない** (どちらに絞ってもサンプルサイズだけ減って Sharpe 悪化)。

これは以下を意味する:
- 「静かな環境の大口買いが続伸する」という仮説は誤り
- 大口買いプリント自体が既に「機関フローの存在」をシグナルしており、追加フィルタは不要
- SBG・MUFG という銘柄選択がもっとも効くフィルタ

## 解釈: なぜSBGがこんなに効くのか

1. **流動性の高さ**: 1日 5.4M ticks → 大口プリントが多く統計力がある
2. **機関投資家の参加**: 海外ファンド・国内機関の主戦場、ポジション転換が頻繁
3. **テーマ株でのフロー**: AI・半導体テーマで機関の建玉転換が大きく振れる
4. **値動きの素直さ**: 流動性が高いので一度方向が決まると続きやすい

逆にトヨタ・アドバンテストが効かない理由:
- HFTの裁定取引が大半 → 大口プリントもアルゴ生成
- 在庫調整が多く方向性なし

---

## 限界・注意点

1. **4ヶ月の限定的サンプル**: レジーム変化への耐性は不明
2. **2026-03 の負け月**: 静穏相場で機能しないリスクがある
3. **1銘柄特化**: SBG が業績・為替・指数イベントで急変したら戦略破綻
4. **コスト4bps**: SBG は流動性高くてスプレッド狭いので妥当だが、執行レイテンシ次第で実コストは1.5倍以上になりうる
5. **大量シグナル (1日147回)**: 自動化必須、手動執行は不可能
6. **Sharpe=18 は1分bar基準**: 日次Sharpe換算では 4-5 程度。それでも非常に高い
