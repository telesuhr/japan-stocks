# 非鉄金属3銘柄 イントラデイ典型パターン分析

**対象**: 5711 三菱マテリアル / 5706 三井金属 / 5713 住友金属鉱山
**期間**: 2025-04-01 〜 2026-04-20 (258営業日, 1分足)
**目的**: 非鉄3銘柄のイントラデイにおける典型的な時間帯別挙動を把握し、エントリー/エグジットの設計に活かす

---

## エグゼクティブサマリー

| 項目 | 5711 三菱マテリアル | 5706 三井金属 | 5713 住友金属鉱山 |
|---|---|---|---|
| 9:00→15:30 累積ドリフト | +18bps | +47bps | +17bps |
| ピーク到達時刻 | 15:30 (引け) | 15:23 (引け直前) | 09:09 (寄り直後!) |
| ボラ最大時刻 | 9:00 (96bps) | 9:00 (115bps) | 9:00 (110bps) |
| ボラ最小時刻 | 11:30 (4.5bps) | 11:30 (6.5bps) | 11:30 (5.5bps) |
| 昼休みジャンプ中央値 | 0bps (std 21) | 0bps (std 32) | 0bps (std 29) |
| 高値形成 9:00-9:30 | 44% | 43% | 45% |
| 安値形成 9:00-9:30 | 48% | 47% | 43% |

**共通して観察された典型パターン**:
1. **U字型ボラ**: 9:00 >> 12:30 >> 15:30 >> 昼前・後場中盤、11:30はボラ最小
2. **出来高は引けに集中**: 15:30単分足が1日の9-16%を占める (5711=16%, 5713=15%)
3. **高値/安値の約45%が寄付30分で決まる** → 日中レンジは序盤で形成されやすい
4. **昼休みジャンプはほぼゼロ中央値** → 後場寄り"飛び"は平均的には無視できる
5. **first30→rest の線形相関はほぼゼロ** (|corr|<0.05) → 単純なモメンタム/リバーサルは効かない

**銘柄ごとの個性**:
- **5711**: 最も素直。first30リバーサル弱、引けドリフト+18bpsを素直に取りに行ける
- **5706**: 最も"動く銘柄"。EODドリフト+47bpsが突出、ボラも一番大きい。first30>+50bpsなら**rest-11bps (リバーサル)**、first30<-50bpsなら**rest+25bps (リバーサル)** ← **明確なリバーサル**
- **5713**: 9:09にピーク(寄り付き買い)→以降レンジ。ピークから引けまで実質横ばい。first30がモメンタム寄り

---

## 1. 9:00基準 累積ドリフト

```
5711: peak 15:30 +18.4bps, trough 09:00 -7.0bps, EOD +18.4bps
5706: peak 15:23 +47.1bps, trough 09:00 -2.0bps, EOD +47.1bps
5713: peak 09:09 +18.4bps, trough 09:00 -2.2bps, EOD +16.7bps
```

- **5711/5706 は"引け上げ型"**: 後場後半にかけて水準切り上げ、引けで最高値
- **5713 は"寄り付き完結型"**: 9:09がピーク、以降は終日レンジで推移
- コスト8bps (LS往復×2銘柄) を考えても、5706は+47bpsで40bps弱のα余地あり

**実装ヒント**:
- 5706を Long、5713を Short (寄り付き直後)→ 引けで手仕舞い
  - 期待値: 5706 +47bps - 5713 +17bps ≒ +30bps/日 - 8bps = +22bps (片側1000万で+2.2万円)

---

## 2. 時間帯別 1分足ボラティリティ (bps std)

U字型だが左端が右端より明確に大きい (寄り付き優位):

| 時刻 | 5711 | 5706 | 5713 |
|---|---|---|---|
| 09:00 | **96.1** | **115.2** | **110.1** |
| 09:15 | 17.2 | 37.1 | 24.9 |
| 09:30 | 15.8 | 24.6 | 19.4 |
| 10:00 | 12.6 | 20.7 | 17.9 |
| 11:00 | 9.5 | 15.0 | 12.1 |
| 11:30 | 4.5 | 6.5 | 5.5 |
| 12:30 | 38.9 | 62.0 | 50.8 |
| 13:30 | 6.8 | 10.2 | 12.0 |
| 14:30 | 8.5 | 12.4 | 14.3 |
| 15:30 | 25.6 | 36.4 | 32.2 |

- **9:00分足が突出** (他時間の5-10倍) → 寄り付き成行は不利
- **11:30が最小**、指値執行にはここが有利
- **12:30(後場寄り)も大きい**: 昼休み情報の織り込み

## 3. 出来高プロファイル

15:30クロージング・オークションが圧倒的:

```
5711: 15:30=16.12%, 09:00=6.21%, 12:30=2.14%
5706: 15:30= 9.02%, 09:00=4.92%, 12:30=2.01%
5713: 15:30=14.62%, 09:00=6.82%, 09:03=2.50%
```

- 引け単分足だけで**日量の9-16%** → 引け成行の市場インパクトは限定的
- 寄付は5-7% → 市場インパクトは引けの約半分

---

## 4. 寄付30分リターン → 残り時間

|corr|はすべて0.05未満で線形関係は**極めて弱い**。ただし閾値ベースで非対称性あり:

| 銘柄 | first30>+50bps → rest | first30<-50bps → rest | 解釈 |
|---|---|---|---|
| 5711 | +1.9bps (WR 54%) | -18.7bps (WR 48%) | 弱モメンタム (下がり続ける日はある) |
| **5706** | **-11.0bps** (WR 51%) | **+25.4bps** (WR 57%) | **明確なリバーサル** |
| 5713 | +15.5bps (WR 54%) | -18.0bps (WR 44%) | モメンタム |

**実装ヒント**: 5706 はギャップ反転狙いの候補
- 9:30時点で +50bps 以上 → Short 14:30引け前手仕舞い (期待-11bps)
- 9:30時点で -50bps 以下 → Long 引け手仕舞い (期待+25bps, WR 57%)

---

## 5. 昼休みジャンプ (前場引け → 後場寄り)

```
5711: mean=+1.4bps, median=0, std=21, |jump|>20bps=22.9%
5706: mean=-0.7bps, median=0, std=32, |jump|>20bps=40.3%
5713: mean=-0.6bps, median=0, std=29, |jump|>20bps=33.3%
```

- **中央値ほぼゼロ** → 平均的には後場寄りに特殊なバイアスなし
- 5706は40%の日で|jump|>20bps → 昼休み中のマクロイベント感応度が高い
- 持ち越しコストを避けるなら昼休み解消候補

---

## 6. 引け30分 (15:00-15:30)

```
5711 close30 mean: +3.6bps (up日+4.7/dn日+2.3)
5706 close30 mean: +5.1bps (up日+6.7/dn日+3.0)
5713 close30 mean: +3.3bps (up日+6.8/dn日-1.2)
```

- **3銘柄とも引け30分は平均+3〜+5bps のバイアス** (クロージング・オークション前のポジション整理)
- 5713は"朝〜15:00 up"の時のみ+6.8bps、down日は-1.2bpsで**追随しない**
- 15:00 Long → 15:30引け決済の"ラスト30分スワップ"戦略は3銘柄とも期待値プラス

---

## 7. 寄付ギャップフィル率

| 銘柄 | Up-gap フィル | Down-gap フィル |
|---|---|---|
| 5711 | 46% | 50% |
| 5706 | 49% | 58% |
| 5713 | 42% | 52% |

- **Down-gap のほうがフィル率が高い** (5706=58%, 5713=52%) → 下げギャップは日中に埋まりやすい
- Up-gap フィル率は40%台 → 上げギャップは埋まりにくい ("窓開け強気"の地合い)

**実装ヒント**: Down-gap 寄り買い → 引けフィル狙いは成立余地あり (特に5706)

---

## 8. 高値・安値の到達時刻

3銘柄とも酷似:

```
高値: 9:00-9:30 で 43-45% (圧倒的)
      14:00-15:00 で 17-19%
安値: 9:00-9:30 で 43-48% (圧倒的)
      14:00-15:00 で 13-16%
```

- **日中レンジの上下端は9:30までに約45%決まる**
- 寄付30分のレンジがそのまま日中レンジになる確率が高い → **9:30時点のOR (Opening Range) を基準にしたブレイクアウトは有効**

## 戦略候補 (順位付け)

| 優先度 | 戦略 | 対象 | 期待値/日 | コスト後 |
|---|---|---|---|---|
| ★★★ | 9:30 reversalエントリー (5706, first30<-50bps Long / first30>+50bps Short) | 5706 | +18bps | +10bps |
| ★★ | Down-gap 寄り買い → 引けフィル | 5706, 5713 | +15bps | +7bps |
| ★★ | 15:00 Long → 15:30 決済 (ラスト30分) | 3銘柄 | +4bps | -4bps (コスト厳しい) |
| ★ | 9:09 ピーク売り (5713 Short @9:09 → 15:30決済) | 5713 | +2bps | -6bps (ほぼ0) |
| ★ | 5706 Long × 5713 Short (9:30→15:30) | ペア | +30bps | +22bps |

**結論**: **5706の"first30 reversal"が単独戦略として最も期待値が高い**。5706は非鉄3銘柄の中でも最も"動く"銘柄で、寄付オーバーリアクションが日中に巻き戻る傾向が明確。

---

## 注意事項

- 期間は2025-04〜2026-04 の1年分。イラン戦争 (2026-02-28) を含むため、戦争前後で挙動が異なる可能性あり (別途 `20260422_dayofweek_nonfer_semi` 参照)
- first30→rest の相関は線形では弱い。非線形・閾値ベースで評価すること
- コスト4bps (アウトライト) / 8bps (LS) で計算。実運用では滑り・金利を+2-4bps 追加想定
- N=258日なので日次戦略の t-stat は十分だが、条件分岐後は N=60-100 に縮小する
