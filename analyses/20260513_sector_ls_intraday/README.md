## 解釈

### なぜAI半導体はモメンタム支配なのか?
1. **テーマ性が強い**: AI/EUV/HBM等の話題で銘柄群が一斉に動く
2. **個別ニュースの効果が大きい**: 決算・受注・指数組み入れで一日中続く方向
3. **海外連動性**: NVIDIA等の米国半導体株の動きを朝に織り込み、その流れが続く

### なぜ非鉄は微弱 Reversion なのか?
1. **LME銅価格に共通連動**: 朝の乖離は需給ノイズ → 引けまでに収束
2. **製錬業ファンダが類似**: 銅亜鉛価格、為替が共通要因
3. **電線3社 (5801-5803) はやや独自要因**: フジクラはAI/光通信寄りで乖離

### ソシオネクスト-ローム ペアが有望な理由
- **ソシオネクスト**: 設計ファブレス、車載中心
- **ローム**: パワー半導体、自社製造
- ビジネスモデルが大きく異なり、朝の同方向の動きはノイズ → 引けに収束

## 実装可能な戦略

### Strategy SL1: AI半導体 Best-Worst Momentum

```
銘柄: ディスコ・ソシオ・アドバン・レザテク・ローム・太陽誘電・東エレ
判定時刻: 10:00
ロジック: 寄付→10:00 で最強銘柄をロング, 最弱銘柄をショート
保有: 10:00 〜 15:30 (大引け)
コスト: 8bps (2銘柄分の往復)
期待: +9bps/日, Sharpe 0.65, 勝率 56%
```

### Strategy SL2: ソシオネクスト-ローム ペアリバージョン

```
ペア: 6526 - 6963
判定時刻: 10:00
発動条件: |寄付→10:00 スプレッド| > 50bps
ロジック: 大きい方をショート, 小さい方をロング
保有: 10:00 〜 15:30
期待: +12bps/トレード, Sharpe 0.91, 年間~190回
コスト後: +12 - 8 = +4bps
```

### Strategy SL3: 三菱マテ-住友鉱山 ペアリバージョン

```
ペア: 5711 - 5713
判定時刻: 10:00
発動条件: |スプレッド| > 50bps
保有: 引けまで
期待: +5bps/トレード, Sharpe 0.71
```

### Strategy SL-Combo: 上記3戦略の合成

各日のシグナル発生に応じて: SL1 + SL2 + SL3 を並行運用。  
仮にすべて同時実行できれば Sharpe 1.0-1.5 程度になる可能性。

## 限界・注意点

1. **Sharpe絶対値は決して高くない** (0.5-0.9): ティック戦略 (Sharpe 18) とは桁違い
2. **大引け持ち越し**: 引け成行の流動性・価格にスリッページ発生
3. **コスト 8bps の妥当性**: 流動性高い銘柄ばかりなので妥当だが、引け15:30の薄板で約定にズレが出る
4. **2024-2025 が悪い**: 直近の頑健性は要OOS検証
5. **AI半導体のMomentum は変動が激しい**: 大きく勝つ日と大きく負ける日の繰り返し
