# LME銅 → 日本株 オーバーナイト戦略

**実施日**: 2026-04-21
**対象期間**: 2025-04-01 〜 2026-04-21 (約13ヶ月, シグナル有効日数 263日)

## 戦略ロジック

1. **シグナル計算**: LME銅(CMCU3)のオープン〜東京引け直前(JST 15:25)の値動き幅
   - 夏時間(UK BST, 3月末〜10月末): LMEオープン = JST 9:00
   - 冬時間(UK GMT): LMEオープン = JST 10:00
2. **エントリー**: |値動き| > 閾値 なら日本株を引け(15:30)で同方向エントリー
3. **決済**: 翌営業日の寄付(9:00)でクローズ
4. **コスト**: 片側2bps × 往復 = 4bps

## データ品質問題 (重要)

`detect_outliers.py` で|overnight return|>20%のトレードを検出した結果、以下の**データ調整不整合**が判明:

- **7012 川崎重工**: 2025-04-22〜07-02の約60日間、close→翌open が毎日 +400% ジャンプ (5:1株式分割の片側調整漏れ)
- **8001 伊藤忠**: 同様に2025-04-22〜06末 +400% (5:1分割片側漏れ)
- **8830 住友不動産**: 2025-05-12〜07-03 +100% (2:1分割片側漏れ)
- **9984 ソフトバンクG**: 2025-12-26に-74.8% (4:1分割)
- その他 285A.T, 4183.T, 5803.T など単発の大型分割

→ `|overnight return| > 15%` のトレードを全て除外することで解決 (91トレード除外)。

## 結論

1. **LME銅のイントラデイ動きは、日本株の翌営業日始値に対して有意な予測力を持つ**
2. 効くのは**景気敏感+グローバルエクスポージャー銘柄**（非鉄、半導体、重工、海運、不動産、エネルギー、外需型医薬）
3. **内需・独自ドライバー銘柄（小売、鉄道、ゲーム）は逆に動く** → 空売り戦略の候補
4. **閾値1.0%が最適**: 年間36回のシグナル、エッジ大
5. **データ調整不整合が広範に存在** — 株式分割後のopen/close不整合に注意
