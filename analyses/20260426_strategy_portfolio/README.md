## 総合評価

1. **ST1は停戦後に最強**: Post期間で Sharpe+18.69。「戦争が終われば LME→株の効果が消える」という懸念は杞憂だった
2. **ST2は安定継続**: Post N=4で判断力は弱いが、方向性は維持
3. **複合Sharpe+8.11 / MaxDD-327bps**: 個別戦略より安定しており、3戦略の組み合わせが有効なポートフォリオを構成
4. **2025-05 の負け月はほぼゼロ (-2bps)**: 実質的に全月プラスという驚異的な安定性

## 限界・注意点

1. **ST3 Post N=3**: 評価不可。今後の木曜LME発動日で要観察
2. **戦略間相関 N=9〜10**: 相関推定値の信頼区間が広い
3. **複合ポートフォリオの資本配分**: 戦略間で「何対何」で資本を配分するかは未定義 (現在は等量)
4. **ST2 信用コスト未計上**: ショートの金利コストを差し引くと月換算で数bps減
