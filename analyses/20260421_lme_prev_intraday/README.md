# LME前日終値変化率 → 翌日日本株イントラデイ戦略バックテスト

分析日: 2026-04-21

## 戦略概要

**シグナル**: Day N の LME銅(CMCU3) close_price の前日比変化率  
**エントリー**: Day N+1 の日本株 9:00 寄付（始値）  
**決済**: 9:30 / 10:00 / 11:30 / 13:00 / 15:30 の各時点  
**方向**: Long（LME上昇→買い）/ Short（LME下落→売り）両方検証  
**コスト**: 往復4bps  

既存の `lme_on_copper` 戦略（当日LME東京時間変化 → 当日引けエントリー → 翌朝決済のオーバーナイト）とは異なる、純粋イントラデイ戦略。

## 主要発見

### 1. 強い順張りモメンタム効果
- **Long（LME上昇→日本株買い）** がすべての閾値・決済時刻で一貫して正のSharpe
- **Short（LME上昇→日本株売り）** はほぼ全パターンで負のSharpe → 逆張りは機能しない
- バスケット結果を見ると Long は上位10がすべてプラス、Short は下位20に集中

### 2. 最強銘柄は住友金属鉱山（5713.T）
- threshold=2.0%, Long, exit=10:00 で Sharpe=21.77（ただしn=5）
- threshold=1.5% まで広げても Sharpe=8〜9 を維持（n=15）
- 非鉄金属株がLMEシグナルに最も感応することを確認

### 3. サンプル数の問題（重要）
- threshold=2.0% では n=5 と非常に少ない → 統計的信頼性が低い
- threshold=1.0% では n=37 程度で一定の信頼性
- **実用的には threshold=1.0〜1.5% が妥当**

### 4. threshold=1.0% Long バスケットでの安定性確認
- exit=15:30: Sharpe=5.33, mean=+0.34%, WR=59.5%, n=37
- exit=10:00: Sharpe=4.14, mean=+0.17%, WR=64.9%, n=37
- 全決済時刻でプラスを維持（PF > 1.4）

### 5. 散布図の相関
- LME前日変化率と翌日寄付〜各時点リターンに正の相関あり（fig3参照）
- 相関は決済時刻が長いほど強まる傾向（モメンタムの持続）

## 懸念事項

1. **サンプルサイズ**: threshold=2.0% は n=5 で過剰適合の可能性が高い
2. **スリッページ**: 寄付の始値での約定を前提とするが、実際は差異あり
3. **データ期間**: 株式1分足が349日と限定的（LMEデータ1265日に対して27%のみ対応）
4. **武田（4502.T）のShort**: LME上昇時に短期逆張りが機能する理由が不明確

## 結論

LME銅前日変化率は翌日日本株（特に非鉄金属株）の寄付からの方向に**統計的に有意な順張り効果**を持つ。コア5銘柄バスケットで threshold=1.0%, Long では Sharpe=4〜5 程度（n=37）を達成。住友金属鉱山が最も感応度が高い。ただし threshold=2.0% の高Sharpe（>10）はサンプル数n=5のため実用性は低い。実運用には threshold=1.0〜1.5% での検証継続と、より長期のデータ蓄積が必要。
