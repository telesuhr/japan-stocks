# 銘柄バスケット定義メモ

`mdutil.py` で定義している銘柄バスケットの意図・選定理由・使い方を整理。

---

## CORE5 — 景気敏感主要5銘柄の等加重バスケット (本命)

| コード | 銘柄名 | セクター | 備考 |
|---|---|---|---|
| **5711.T** | 三菱マテリアル | 非鉄金属 | LME銅との連動が最も強い |
| **6501.T** | 日立製作所 | 総合電機 | 景気敏感の代表格、流動性◎ |
| **7011.T** | 三菱重工業 | 重機・防衛 | 防衛需要+景気サイクル |
| **5016.T** | 出光興産 | 石油・エネルギー | 原油/製品マージン感応 |
| **4502.T** | 武田薬品 | 医薬品 | ディフェンシブでβ調整役 |

### 選定意図
もともと LME銅オーバーナイト戦略 (Sharpe+11.5) の検証時に
「LME銅が+1%上げた翌朝、日本株でどれを買うか」を決める必要があり、
以下の条件で絞ったもの:

1. **景気敏感 (LME/原油との連動)**: 三菱マテリアル・出光・日立・三菱重工
2. **流動性が高い** (スリッページが小さい, 売買代金が大きい)
3. **セクター分散** (非鉄1・電機1・重機1・エネ1 + ディフェンシブ1)
4. **マニュアル発注で追える銘柄数 = 5つ** に絞った

### 注意点
- 武田(4502) はディフェンシブなので LME/原油との連動は弱め。バスケットのβを下げる役割
- 厳密な「景気敏感コア」だけなら 武田を外して 4銘柄+半導体1 (TEL等) にするほうがクリーン
- 曜日分析で **コア5 火曜-48bps(t=-3.3)** という強いシグナルが出たのは、
  三菱マテ・日立・三菱重工・出光の4銘柄が揃って火曜寄り天だから (武田は寄与小)

### 主な登場戦略
- 戦略本命: **LME銅+1% → コア5Long** (Sharpe+11.5)
- 戦略7: **LME+1% → LS (コア5 Long + DOMESTIC_SHORT Short)** (Sharpe+11.2)
- 戦略4/5: **日経先物/TOPIX夜間+0.3% → コア5Long** (Sharpe+4.7)

## 使い分けフロー

```
シグナル → バスケット選択
────────────────────────────
LME銅+1%           → CORE5 (本命) or NONFERROUS (純血)
LME銅+1% + ヘッジ  → CORE5 Long + DOMESTIC_SHORT Short (LS)
WTI+0.5%           → ENERGY
Brent+2%           → SHIPPING
日経/TOPIX夜間上げ → CORE5 (流動性+分散)
水曜寄り (曜日)    → NONFERROUS + ENERGY + SEMICON
火曜寄り天Short    → CORE5 (非鉄・重厚長大が寄り天になりやすい)
月曜寄り Short     → SHIPPING (最弱曜日)
```
