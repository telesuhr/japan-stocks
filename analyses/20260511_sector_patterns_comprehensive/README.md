**分析日**: 2026-05-11
**対象**: プライム1,413銘柄 × 33セクター × 2024-01-04〜2026-05-08
**サンプル**: 365,254 銘柄日（売買代金5億円以上フィルタ）

---

## 分析の目的・仮説

新DBスキーマで全プライム銘柄の2年分データが揃ったため、**日次・ON・イントラデイ・曜日・モメンタム/リバーサル・出来高感応度・時間帯ボラ**の7軸で各セクターの特徴を体系的に洗い出す。トレーディング戦略の母集団を作る基礎調査。

### 📊 ON特性 — ギャップフィル率

**GUフィル率（ギャップアップ後に寄付きを下抜けた率）TOP3**:
1. 電気・ガス業 28.6%
2. 海運業 28.2%
3. 情報・通信業 26.7%

**観察**: ディフェンシブ・運輸系は**ギャップが埋まりやすい**。逆に金属製品・銀行・繊維などはギャップ持続性が高い（フィル率20-22%）。

→ **ON戦略の銘柄選定**: ギャップフィル率が低いセクター（銀行・金属製品）の方がON系戦略に向く。

## トレーディング戦略への示唆

### 既存戦略の補強候補
| 既存戦略 | 示唆 |
|---------|------|
| `bank_absorption` | 銀行業の強リバーサル特性（-0.086）と整合。戦略の経済的妥当性確認 |
| `lme_on_copper` | 非鉄金属はON寄与56%・Sharpe1.54。ON戦略との親和性高い |
| `topix_overnight` | 全セクターで月曜最悪・火曜最良 → 月曜引け→火曜寄付きの追加検証価値 |
| `orb_breakout_long` | 電気機器・非鉄金属の9:00-9:15ボラ最大 → ORBユニバース拡張候補 |

### 新規戦略候補（次の研究テーマ）
1. **「月曜引け→火曜寄付き ON Long」**全セクター対応版（30/33で月曜最悪・火曜最良）
2. **金属製品の出来高順張り**（vol≥3x → 翌日+1.28% / 低出来高比+1.26%）
3. **石油・石炭・パルプ紙の大商いSHORT逆張り**（vol≥3x → 翌日-0.85〜-0.92%）
4. **非鉄/銀行/建設の5日リバーサル**（自己相関-0.08〜-0.11、bank_absorption拡張）

### 戦略不向きセクター
- **空運業**: ボラ低・Sharpe負・サンプル少 → トレーディング対象外
- **陸運業**: 鈍重で曜日効果も弱い
- **食料品**: 動かない・モメンタム弱い

## データ・手法

### 使用テーブル
- `stocks_daily` (adj_open/adj_close/adj_volume/turnover_value)
- `stocks_intraday` (時間帯ボラ・AM/PM計算用)
- `symbol_master` (sector33_nm)

### 主要パラメータ
- 流動性フィルタ: 売買代金 ≥ 5億円
- ボラ計算: 過去20日（直近2日除外）の出来高MA
- 自己相関: 過去N日リターン × 翌5日リターン

---

## 限界・注意点

1. **AM/PM計算は代表99銘柄のみ**（前段で morning_close が NULL のため）
2. **超急増のN数が少ない**: パルプ・紙 / 鉱業 / 石油 などは大商いの絶対数が少なくロバストでない
3. **2024-2026の特殊環境**: 日銀利上げ・米中関係など特異要因含む。10年スパンの安定性は別途検証要
4. **survivorship bias**: 現在上場のプライム銘柄のみ。上場廃止考慮外

## 次のアクション候補

優先順:
1. **月曜→火曜 ON戦略の全セクター適用検証** → 既存`topix_overnight`の拡張可能性
2. **金属製品の出来高順張り 単独戦略バックテスト**（Sharpe>1.5目標）
3. **エネルギー系の大商いSHORT検証**（石油・パルプ・鉱業の組み合わせ）
4. **電気機器の9:00-9:15 ORB戦略**（既存`orb_breakout_long`の電機版）
