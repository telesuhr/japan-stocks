## 主要発見

### A. 短期（1〜5分）は平均回帰、ORBは強いモメンタム

| パターン | 継続率 | 解釈 |
|---------|--------|------|
| 1分足 Lag1 自己相関 | r=-0.012 (p<0.001) | 強い平均回帰 |
| 5分後継続率（中幅 0.1-0.3%） | 44〜45% | 平均回帰傾向 |
| **ORB上抜け（8:45-9:15レンジ）** | **94%** | 圧倒的モメンタム |
| **ORB下抜け** | **89%** | 同上 |

→ **短期の小幅な動きは逆張り有利、ORBブレイクは順張りが適切**

### B. 10時前後でレジームが変わる（ユーザー仮説を定量確認）

| 時間帯 | 5分後継続率 | 傾向 |
|--------|------------|------|
| 8:55 | 51.4% | わずかにモメンタム |
| 9:00-9:15 | 48-50% | ほぼ中立 |
| 9:10 | 45.0% | 平均回帰始まる |
| **10:00** | **41.9%** | **← 急落** |
| 10:30 | 41.6% | さらに強まる |
| 15:00 | 37.7% | 最強の平均回帰 |

**9:30-10:00 の動きの10:00以降反転率: 57%**  
**9:00-10:00 → 10:00-11:30 の方向予測力: r=+0.003（ゼロ）**

### C. 最大ボラは9:00ではなく12:30（後場寄付）

| 時間 | 1分足ボラ |
|------|---------|
| **12:30** | **0.202%** ← 最大 |
| 9:00 | 0.144% |
| 9:01-9:06 | 0.10〜0.12% |

### D. 平均価格経路: 9時が安値 → 14:40がピーク

- 「寄付後に下押し → 午後にかけて緩やかに上昇」が平均的パターン
- 9:00〜9:05 付近が当日安値になりやすい

### E. 大引け前30分は反転ゾーン

- 14:45時点の損益方向と大引け方向一致率: **40%**（反転率60%）
- その日プラスだと引け前に売られ、マイナスだと買い戻される傾向

### F. 後場寄付（12:30）はギャップ継続型

- 後場ギャップ → 後場最初30分の方向一致: r=+0.131（有意 p=0.032）
- 前場のトレンドより「12:30の値決めの勢い」を引き継ぐ

### G. 寄付ギャップ分析

| パターン | 反転率（ギャップフィル） |
|---------|----------------------|
| 小GU (0〜0.3%) | 59% |
| 中GU (0.3〜1%) | 44%（継続優勢） |
| **大GD (<-1%)** | **75%（強い反転）** |

### H. 曜日効果

| 曜日 | 平均リターン |
|------|------------|
| **火** | **-0.277%** ← 唯一の負 |
| 金 | +0.199% |
| 水 | +0.182% |
| 月 | +0.101% |
| 木 | +0.032% |

```
時間帯    レジーム        推奨アプローチ
───────────────────────────────────────────────────
8:45      高ボラ起動       様子見 or ORBレンジ確定待ち
9:00-9:15 ORBウィンドウ   レンジ形成を観察
9:15以降  ORBブレイク判定  上下どちらかに抜けたら順張り（継続率89-94%）
9:10-10:00 平均回帰始まる  小幅な動きには逆張りを検討
10:00以降 平均回帰強まる   継続率41%台、スキャルプ逆張り有利
12:30     最大ボラ         後場ギャップの方向に乗る
14:45以降 反転ゾーン       当日損益方向と逆に動きやすい（60%）
```
