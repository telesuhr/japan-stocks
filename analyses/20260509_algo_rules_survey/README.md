# 幅広アルゴリズムトレーディングルール検証

## 分析の目的・仮説

日本個別株で「何らかのルール」が機械的に機能するか、7カテゴリー51ルールを一括検証する。  
ルック・アヘッドバイアスを防ぐため、**特徴量と目的変数の時間的分離**を厳守。

| 目的変数 | 内容 | 使える特徴量 |
|---|---|---|
| `aft_ret` | 後場リターン(後場寄付→大引, 12:30~15:00) | ギャップ・前場情報・ORB |
| `on_ret` | オーバーナイト(当日引け→翌日寄付) | 当日全データ |
| `next_fullday_ret` | 翌日全日リターン | 当日全データ |

---

## データ概要

- 期間: 2025-01-01 〜 2026-05-07 (約340営業日)
- 銘柄: 日本株117銘柄 (半導体・非鉄・海運・自動車・金融・商社・通信・内需等)
- データ: 1分足 約1,161万行
- 日次レコード数: 36,318件 (117銘柄 × 約310日)
- コスト前提: 片側2bps × 往復 = **4bps**

## 詳細解釈

### 1. 後場リターン系 — 小幅だが頑健

**後場での取引可能なルールは存在するが、エッジは小さい。**

| ルール | 解釈 |
|---|---|
| C3: 前場弱 → 後場買い (+1.5bps) | ランチタイムの売り圧力緩和後に小幅回復。日本市場の典型的パターン |
| A5: ギャップダウン大 → 後場買い (+1.9bps) | 大ギャップ後の部分的ギャップ埋め。過売られ後の反発 |
| G2: 前日弱×ギャップダウン → 後場買い (+5.5bps) | 2段階の弱さ後に反転。最も明確な後場エッジ |

**負けルール (後場):**
- C4/C2: 前場弱/強 → 後場売り方向は負け (日本株は後場に小幅リバーサルが常態)
- D2/D4: ORB下抜け後の後場売り (-9.7bps) — ORBブレイクダウンは後場で逆行

### 2. ON (オーバーナイト) リターン系 — 数値は外れ値依存

ON系ルール (B6, A8) の平均リターンが300-500bpsと異常に大きい。  
これは **2025年4月の関税ショック局面**で株価が複数日連続下落 → 翌日大幅ギャップアップ、という極端なイベントが平均を大きく引き上げているため。

```
6857.T ONリターン分布:
  平均: +20.8bps, std: +339.5bps
  5%ile: -497bps, 95%ile: +438bps
  |ON|>200bps: 130件 / 325件 (40%)
```

**外れ値を除いた信頼できるONルール:**
- **F4: 前日弱×ギャップダウン → ON買い (+34.4bps, std=284bps, t=7.64)**  
  これは合理的な範囲内。2日連続下落後の翌朝リバーサルを捉えている。

### 3. 翌日リターン系 — 驚きの発見: モメンタム継続

- **B4: 前日弱(-100bps以下) → 翌日売り (+93.5bps net, t=10.04)**  
  → 前日の大幅下落後に翌日も下落が続く (リバーサルではなくモメンタム!)
- **A6: ギャップアップ大 → 翌日売り (-73.9bps)** = つまり翌日も下落
  
**日本株は1日スケールでは mean reversion ではなくモメンタム継続型。**  
ただし関税ショック局面の連続下落が統計を歪めている可能性あり。

### 4. 機能しないルール

| カテゴリー | 結果 |
|---|---|
| ORB ブレイクアウト (後場) | すべて負け方向 — ORBブレイクは後場で反転 |
| 前場強 → 後場モメンタム | 負け (C1: -1.1bps) |
| 出来高急増系 | N少なく統計的に不安定 |
| ボラ収縮後ブレイク | 有意な正エッジなし |
| 前日前場強 → 翌日買い | -23.8bps (逆効果!) |

## 限界・注意点

1. **関税ショック期間の影響**: 2025年4月に多くの株が急落→急回復。ON系リターンはこの期間に大きく歪む
2. **N不足の複合ルール**: G1, E6, E7 等は N<200 で統計的パワー不足
3. **銘柄間の非独立性**: 同日同市場で動く銘柄は独立サンプルではない → 実効Nは名目Nより小さい
4. **後場aft_retの小さいエッジ**: 1.5-5.5bps は実際のスリッページ (1-3bps) を考慮すると薄い
5. **多重比較問題**: 51ルールを検証しているため、Bonferroni補正後 p<0.001 を有意基準にすべき（p<0.05では偶然有意が2-3ルール出る）
