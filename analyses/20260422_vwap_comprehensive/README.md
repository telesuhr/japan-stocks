## 4. Attraction テスト — VWAPに引き寄せられるか？

**手法**: 毎分 |dev| が閾値 X を横切った瞬間を記録 → 60分以内に価格が VWAP (|dev|<5bps) に戻った割合を計算。

### 閾値50bps / 60min以内戻り率 (上位・下位)

| 銘柄 | up→戻り% | dn→戻り% | 対称性 |
|---|:-:|:-:|---|
| **高い (引き寄せ強)** | | | |
| 6857 アドバンテスト | 39.1 | **60.2** | 下乖離強く戻る |
| 6146 ディスコ | 42.0 | 38.1 | 対称 |
| 5016 出光 | 43.4 | 39.2 | 対称 |
| 5706 三井金属 | 41.7 | 43.2 | 対称 |
| **低い (引き寄せ弱 = Breakout化)** | | | |
| 4502 武田 | 18.8 | **13.4** | 小幅銘柄だが戻らず |
| 9020 JR東 | 26.3 | 20.0 | 安定系だが戻らず |
| 9101 日本郵船 | 20.3 | 25.7 | Breakout銘柄 |
| 9104 商船三井 | 18.4 | 29.3 | 非対称 |

### 閾値80bps → 戻り率の減衰
| 銘柄 | th=30 | th=50 | th=80 |
|---|:-:|:-:|:-:|
| 6857 アドバン up | 52.4 | 39.1 | 24.4 |
| 5706 三井金属 up | 54.2 | 41.7 | 31.3 |
| 9101 日本郵船 up | 34.5 | 20.3 | 13.5 |

**解釈**: 30→80bpsの閾値アップで戻り率が半減。「大きく離れた乖離は戻らない」= 閾値が大きいほどBreakout挙動が支配的。

### 中央値戻り時間
概ね **13〜25分**。つまり戻るとしたら短時間で戻る。20分超えて戻らなければその日は戻らない可能性が高い。

## 9. ポートフォリオ観点

### 独立性・多様化
4タイプ (Trend/Reversion/Breakout/Slope) は**発動条件が異なる**ため、単日に複数トレード発生する可能性は低い:
- 朝Trend (5020, 8411, 4063 Slope): 9:30-10:00にシグナル
- Reversion (5016, 8035): 10:00-11:30のレンジ後
- Breakout (9101): 60分以上待つので早くて10:00以降

→ **日次では互いにほぼ直交**。全銘柄同時採用でリスク分散可能。

### セクター分散
5020 エネルギー / 9101 海運 / 4063 半導体 / 5016 エネルギー / 8411 金融 / 8035 半導体 — 金融・海運・エネルギー・半導体の4セクターに分散。

### コスト感応度
全合格戦略は mean >= +17bps でコスト4bps後。倍のコスト8bpsでもSharpeは2/3維持、実運用耐性は十分。

## 10. 限界と注意事項

- **一部 t-stat < 2 の戦略が含まれる**: Reversion Top や Slope Top の一部は Sharpe が高くとも t<2.0 で統計有意性が弱い。本文で ✓ 印を付けた戦略のみが厳格合格
- **Breakoutは銘柄依存性極大**: 9101 だけで合格 → 1銘柄にアルファ集中、Out-of-Sample でDD発生するとポートフォリオ影響大
- **期間にイラン戦争(2026-02-28〜)を含む**: レジーム変化で性能が変わる可能性。戦争前後分割での安定性検証を推奨
- **Slope は 15分窓固定**: 5/10/30分 など他窓のロバスト性は未検証
- **Attractionテストは日中の全イベント**: 連続クロスを別々に数えているため、同日内で重複カウントの可能性あり
- **過学習リスク**: 22銘柄 × 6時刻 × 5閾値 × 2方向 = 1,320パターンの中のTop。Bonferroni補正すればp値≈0.15で合格は3〜4件に減る可能性
