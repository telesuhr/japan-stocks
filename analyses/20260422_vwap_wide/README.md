## 戦略ロジック

### VWAP乖離（dev）の定義

```
VWAP_t = Σ(close_i × volume_i) / Σ(volume_i)  ← 寄付(9:00)からの累積
dev_t  = (close_t / VWAP_t - 1) × 10000         ← bps単位
```

volumeが0またはNullの場合はclose×1で代替。

### 戦略タイプ

| タイプ | ロジック | 想定する市場特性 |
|--------|---------|----------------|
| **Trend** | dev ≥ +th → Long / dev ≤ -th → Short | 乖離がそのまま継続（モメンタム） |
| **Reversion** | dev ≥ +th → Short / dev ≤ -th → Long | 乖離後にVWAPへ引き戻される |

### グリッドパラメータ

- **signal_time**: 9:30 / 10:00 / 11:00 JST
- **threshold**: 20 / 30 / 50 bps
- **strategy**: Trend / Reversion

### エントリー・エグジット

| タイミング | 詳細 |
|-----------|------|
| エントリー | signal_time の次の1分バーのopen |
| エグジット | 同日 15:20-15:30 の最後のclose |
| 保有時間 | 約4.5〜6時間（純粋イントラデイ） |

## 主要発見

### 1. 「9:30×50bps×Trend」が最強パターン

| 銘柄 | Sharpe | 解釈 |
|------|--------|------|
| 1925.T 大和ハウス | +7.98 | 寄付30分で大きく上乖離 → そのまま引けまで続伸 |
| 9983.T ファストリ | +7.53 | 同上。インデックス寄与度が高く買い圧力が持続 |
| 5020.T ENEOS | +5.56 | 原油動向と連動、朝のモメンタムが強い |

大きな乖離（50bps ≈ 0.5%）が9:30時点で出ている日はそのままトレンド継続する傾向が強い。

### 2. 出光（5016.T）はReversion型の代表

10:00時点の乖離（20/30/50bpsすべて）でReversionが機能。  
Nが最大188と多く統計的信頼性が高い。  
エネルギー株は日中の行き過ぎがVWAPへ引き戻される動きが強い。

### 3. 野村証券（8604.T）はReversionが機能する金融株

Sharpe+4.89、WR62.8%。証券株は日中の値幅が大きく、行き過ぎが修正される。  
9:30×50bpsの大乖離日に逆張り→引けで戻りを取る形。

### 4. JR西日本（9023.T）は3パターン合格の安定Trend型

Nが75〜117と多く、特に10:00×20bps（N=117）は発動頻度が高い。  
内需・ディフェンシブだが日中モメンタムが出やすい銘柄。

### 5. 非鉄金属・半導体は合格ゼロ

lme_on_copper戦略（ONホールド）で優位だった銘柄群がVWAPイントラでは合格しない。  
→ これらのセクターはオーバーナイトの外部シグナル連動が強く、日中のVWAP乖離はノイズ。

## 注意事項

1. **Sharpe年率化**: `mean/std × √252` のper-trade Sharpe。月次発動N=30〜130なのでt-statで有意性を判断すること。
2. **データ異常**: 8001.T・8830.Tは日中価格が5倍/2倍の開きがあり除外必須。データソースの確認が必要。
3. **スリッページ**: 成行でのエントリーを前提。流動性が低い銘柄（5332.T TOTO等）は実際のコストが高い可能性。
4. **Out-of-Sample未実施**: すべて同一期間のin-sample。実運用前に別期間での検証が必要。
