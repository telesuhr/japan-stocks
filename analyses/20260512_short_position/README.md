## 主要発見

### 1. 「空売り急増 → 短期 squeeze」: 弱いが実在 (t=4.34)

| 仕分 | N | mean (5d) | t値 | Sharpe |
|---|---|---|---|---|
| 空売り急増Top20% → ロング | 115,414 | +12.1bps | **4.34** | 0.09 |
| 空売り急増Top10% → ロング | 58,051 | +11.9bps | 2.75 | 0.08 |
| 空売り急増Top5% → ロング | 29,382 | +13.6bps | 2.08 | 0.09 |

→ 統計的有意 (大量N の効果) だが、mean=+12bpsとリターンの絶対値は小さい。  
個別株のリターン分散が大きい (std=944-1125bps) ため Sharpe が低い。

### 2. **【重要】機関のショートカバー → 翌週さらに下落** (t=-5.67)

| 仕分 | N | mean (5d) | t値 |
|---|---|---|---|
| 空売り減少Top5% → ロング | 28,091 | **-33.6bps** | **-5.67** |
| 空売り減少Top10% → ロング | 56,746 | -21.9 | -5.56 |
| 空売り減少Top20% → ロング | 114,172 | -14.7 | -5.63 |

逆方向に読むと:
- **「機関のショートカバー直後 → ショート」: +34bps/5日, t=+5.67**

これは新発見:
- 機関がカバーする = 「もう下落は終わった」と判断
- 実際はその後さらに下落 = **機関の判断は早すぎる**
- 個別株ベースで翌週ショートを仕掛ければ機能する

### 3. Smart Money (ICS, Marshall Wace等) 空売り増 → 弱いがショート方向

| 期間 | N | t値 |
|---|---|---|
| Smart Money 空売り増 → 5日後ショート | 25,753 | -1.52 (非有意) |
| Smart Money 空売り増 → **10日後ショート** | 25,745 | **-1.91 (p=0.056, ほぼ有意)** |

→ Smart Money は短期 (5日) では当たらないが、**10日経過後にやや下落**する兆候。  
ただし統計的検出力は弱い。

### 4. 「空売り蓄積大」「複数機関空売り」は機能しない

- H3 (空売り蓄積 Top10%): t=0.63 (非有意)
- H4 (3機関以上空売り): t=0.38 (非有意)
- H4 (8機関以上空売り): t=-0.11 (非有意)

→ 空売り建玉のレベル (絶対量) は予測力なし。**変化量こそが重要**。

## 解釈: なぜ「機関のカバー」が逆指標になるのか

機関の空売り建玉減少 = 2パターン:
1. **益決め (利益確定)**: 既に十分下げた → カバー (反転買い圧力)
2. **損切り (squeeze被害)**: 上昇に耐えきれず買い戻し (一時的反発でその後さらに上昇)

この両方が混在するが、**集合的には #1 が支配的**。なぜ翌週も下落するか:
- 機関カバー → 短期の買戻し圧力消失 → 残りの売り圧力で下落継続
- カバーするほど下げた銘柄 = ファンダ悪化 → 下落トレンド継続

→ **「機関がカバーした = もう下げない」というナイーブな読みは間違い**。  
むしろ「カバーで一時反発 → その後トレンドフォローで下落」が現実。

## 実装可能な戦略候補

### Strategy S1: ショートカバー逆張りショート (最強の発見)

```
条件: 銘柄ごとの空売り建玉減少率が市場上位 5%
ポジション: その銘柄を空売り
保有: 5営業日
期待: +34bps/5日 (t=5.67 の逆方向利用)
頻度: 1日約150銘柄 × 毎日報告 → 大量シグナル
```

ただし**個別株ショート**にはコストの上乗せ:
- 貸株料 (年1-3%) = 1日 0.03-0.08bps × 5 = 0.15-0.4bps
- 信用売りコスト: 片側2bps × 往復 = 4bps
- 純利益: +34 - 4 - 0.4 = **+29.6 bps/5日**

これは個別銘柄に分散すれば実用レベルになる可能性。

### Strategy S2: Short Squeeze Long

```
条件: 空売り急増 Top10%
ポジション: その銘柄をロング
保有: 5営業日
期待: +12bps/5日 (t=2.75)
コスト後: +12 - 4 = +8bps
```

→ コスト後 net +8bps。Sharpe 0.05 程度。実用には薄い。

### Strategy S3: LS両建て

```
Long: 空売り急増Top10% (squeeze候補)
Short: 空売り減少Top10% (カバー後継続下落)
→ 同時実行で市場ニュートラル化
期待: +12 + 22 = +34bps/5日, コスト8bps差引 → +26bps
```

これは興味深い。**機関の動きの両方を逆に取る戦略**。

## 限界・注意点

1. **個別株リターンの大きな分散**: std=1000bps級 → t値高くてもSharpe低い
2. **報告遅延**: 計算日(calc_date)と開示日(disc_date)に約2日のラグ
3. **公開情報**: 報告は公開なので、既に織り込まれている可能性
4. **0.5% 閾値**: それ未満の空売りは記録されない → ノイズあり
5. **コスト**: 個別株ショートには貸株料・コストが追加で発生
