## 5. 「個別銘柄リスク」の重要性

5/11と5/14の SBG暴落は、市場全体の動きとは無関係に起きている:
- 5/11: SBG -7.65% (市場平均は-0.07%)
- 5/14: SBG また大幅下落 (市場平均は不明、stocks_daily 取得済)

5/14の市場全体: -0.62% (5/12)、5/13は -0.30%。

→ SBG は**市場全体の動きと乖離した独自要因**で動いている可能性大:
- 海外子会社の業績懸念
- 信用買い残の処分
- アクティビスト要因
- AI/テック関連の個別ニュース
