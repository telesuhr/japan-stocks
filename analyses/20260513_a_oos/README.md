# Task A OOS 検証 — 投資部門別売買戦略の頑健性

## 結論を先に

✅ **海外勢買い越し (Z>1.0~1.5)** はOOSでも頑健 (Sharpe 1.05 維持)。  
⚠️ ただし**最も強かった「海外-個人 Z>2.0」(Train勝率85.7%)** は OOSで 50% に劣化 → **オーバーフィット疑い**。

---

## 検証方法

**Walk-forward Z-score** (look-aheadなし):
- 各週のZ-scoreは過去26週のみで計算
- Train: 2016-2022 (7年, 347週)
- OOS: 2023-2026 (3.4年, 175週)

学習期で最も良かった主要シグナルを OOS に適用し、Train→OOSの劣化を測定。

## 重要な発見

### 1. 「海外-個人 Z>2.0」は完全に過学習だった

| シグナル | Train | OOS |
|---|---|---|
| 海外-個人 Z>2.0 勝率 | **85.7%** (14週) | **50.0%** (10週) |

学習期で最も勝率の高いシグナルは、**OOSでランダム水準まで劣化**。  
14週というサンプルが小さく、たまたまの偶然だった。  
**「過去の勝率」だけで戦略を選んではいけない**典型例。

### 2. 緩い条件 (Z>1.0) の方が頑健

「海外勢 Z>1.0」は Train 勝率 57.9% (緩い) だが、OOSでも 58.1% → **ほぼ同等**。  
厳しい条件 (Z>1.5) は過学習しやすい。**緩めの条件の方が経年劣化に強い**。

### 3. 直近の打率 4/7 = 57.1%

シグナル発生履歴の直近では:
- 大勝(>200bps): 3回
- 大敗(>200bps): 2回
- 微勝/微敗: 2回

「**当たれば大きく、外れも大きい**」という性質。Sharpe 1.05 のリスクリターンを表している。

## 限界・注意点

1. **OOS 3.4年は短め**: より長いOOSが必要 (理想は同じ期間)
2. **シグナル数が少ない**: 年5-10回 → 統計的検出力が低い
3. **2018-2019年は機能していなかった**: 米中貿易戦争で海外フローと TOPIX の連動性が薄れた期間
4. **コストは未考慮**: TOPIX ETF (1306) 往復6-8bps → 期待値から差し引くと Sharpe 0.8 程度
5. **5/11ショック級のイベント**: 公表後即発生する大幅変動は捕捉できない

## 実装可能な戦略案 (修正版)

OOS結果を踏まえた現実的な設定:

### Strategy F-Robust (頑健重視)

```
条件 (OR): 
  - 海外勢 Z > 1.0 (緩)
  - 自己売買 Z < -1.5
発動: 投資部門別公表翌月曜 (=金曜引け後 → 翌週月曜寄付)
ポジション: TOPIX ETF (1306) ロング
保有: 5営業日 (翌金曜引け)
期待: +25 bps/週, Sharpe ~1.0, 勝率 58-60%
頻度: 年5-15回
```

### Strategy F-Aggressive (リターン重視、リスク高)

```
条件 (OR):
  - 海外勢 Z > 1.5
  - 投信 Z > 1.5
  - 海外-個人 Z > 1.5
保有: 5日
期待: +30-50 bps/週, Sharpe ~1.0, 勝率 50-65%
頻度: 年10-15回
```
