## 所感
- **火曜・水曜のエッジが突出** (Sharpe +8~13)
- **木曜は戦略全体の足を引っ張る** (LMEシグナル日でもマイナス)
- ベースラインでも木曜は -22bps/日 → 日本株に木曜アノマリー存在
- 金曜は悪くない (三連休前でも継続)

## 運用提案
**木曜日のLMEシグナルは無視する** フィルタを入れるだけで戦略の質が大幅改善:
- 現行 th=1.0% 全曜日: Sharpe+11.5
- 火/水/金のみ: 推定 Sharpe +13以上

シンプルルール: **「LMEが+1%アップ、かつ木曜以外」のみエントリー**
