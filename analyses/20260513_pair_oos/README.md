# セクター LS / ペア戦略 OOS 厳格検証

## 結論を先に

✅ **ペアトレード戦略は OOS で軒並み改善** (6ペア中5ペアが Train > OOS の関係を超えた)。
- Train平均 Sharpe 0.59 → **OOS平均 Sharpe 2.55**
- 「ソシオネクスト-ローム 10:50」が OOS で Sharpe **3.43** に達成
- 過学習どころか **OOSの方が良い** (= 真のエッジがある示唆)

❌ **Best-Worst Spread戦略は全滅** (4/4戦略で OOS Sharpe マイナス)
- AI半導体 BW Momentum: Train +0.96 → OOS **-2.88** (大失敗)

---

## 検証方法

```
Train: 2024-05-10 〜 2025-12-31 (404営業日, 20か月)
OOS  : 2026-01-01 〜 2026-05-19 (88営業日, 5か月)

手順:
  1. Train期間で各戦略の最適判定時刻を探す
  2. その時刻を OOS期間に「そのまま」適用
  3. Train Sharpe vs OOS Sharpe を比較
```

### なぜ OOS で改善したか (考察)

OOSの方が良いというのは異常なので、原因を考える:

1. **2026年は銘柄分散が大きい年**:
   - 5/11ショック等で個別銘柄の動きが大きく出た
   - 同セクター内の乖離 → 引けに収束の度合いが強まった

2. **市場のレジーム変化**:
   - 関税ショック後の高ボラ環境 → 短期トレーダーが減少 → ファンダ通りに動きやすい
   - 機関のロングショートフローが活発化

3. **Train期間の特殊性**:
   - 2024-2025は静かな期間でエッジが薄かった
   - 2026年に入って戦略の前提条件 (収束) が強く成立

ただし OOS期間が **5か月のみ (N=65-73)** と短いので、**過信は禁物**。  
2026後半・2027でも続くかは別途検証必要。

→ **全戦略がOOSで負け**。特に AI半導体 BW Momentum は壊滅的悪化。

### なぜ Best-Worst は失敗したか

1. **Best/Worst 銘柄は変動が大きい**: その日の極端な銘柄を選ぶ → リターン分散がデカい
2. **個別銘柄イベント依存**: ベスト銘柄が決算後の急騰なら、午後もモメンタム継続せず反落することも
3. **2026年は2024-2025と相場性質が違う**: モメンタム/Reversionの軸が反転

## 解釈

### ペアトレードが OOS で機能する理由

**ペア = 2銘柄の相対値の収束**:
- 構造的に「同じセクター・似たビジネス」 → ファンダの差分は限定的
- 朝の一時的乖離は引けまでに収束する性質
- 市場全体の動きにロバスト (LS で市場ベータがゼロに近い)

**ペアの選び方が重要**:
- Train で見つけた最強ペア (ソシオネクスト-ローム等) はビジネスモデルが異なるが同セクター
- → 「動きの源泉が異なる」ため一時的乖離が必ず起きる

### Best-Worst が失敗する理由

**Best/Worst の選択 = 極端値選択**:
- その日の最強・最弱は「個別イベント」起因
- 決算・ニュース・指数組み入れで動いた銘柄はモメンタム/Reversion 両方ありうる
- 単方向 (Reversion or Momentum) では対応不可

→ **「LS のペア」は機能、「上位/下位ピックアップ」は機能しない**。  
これはペアトレードと統計裁定の古典的な原則と整合する。

## 修正された推奨戦略

### Strategy SL-Pair6 (推奨)

OOS結果に基づいて、6ペアすべてを並行運用:

```
判定時刻と運用ペア:
  9:20  - 三井金属-住友鉱山      OOS Sh=3.51
  9:30  - ローム-太陽誘電         OOS Sh=0.77
  10:00 - 三菱マテ-住友鉱山      OOS Sh=3.50
  10:30 - 住友鉱山-DOWA          OOS Sh=3.09
  10:50 - ソシオネクスト-ローム    OOS Sh=3.43
  12:50 - ソシオネクスト-東エレ    OOS Sh=1.01

各ペアで spread > 50bps 時のみエントリー、引けまで保有
コスト: 8bps (LS 2銘柄分往復)

期待 (OOSベース):
  平均 Sharpe ~2.55
  ペア間相関低ければ複合 Sharpe = 3.5+ も可能
  年間トレード数: 250-300 (各ペア4-50回/年)
```

### Strategy SL-BW (非推奨)

OOS で全敗のため**運用しない**。

特に AI半導体 BW Momentum はTrain で見つかった「Sharpe 0.96」というシグナルが完全に過学習だった。

## 限界・注意点

1. **OOS 5か月は短い**: 1年以上のOOSで確証取りたい
2. **OOS が Train を超えるのは普通異常**: 過信せず複数期間で検証必要
3. **2026年は特殊な相場期間の可能性**:
   - 関税ショックの影響残存
   - 5/11級の急落多発
   - 機関フローが活発
4. **N=65-73 トレード**: 統計的検出力はギリギリ
5. **「6ペア合成 Sharpe 3.5+」は理論値**: 実際の相関を加味した計算が必要
