## 2. 日次ペアZスコア平均回帰LS (❌ NG)

ルール: Zスコア(20日/40日) |Z|>1.5 or 2.0 で逆張り → |Z|<0.5 で決済

**全12組合せ中Sharpeプラスはゼロ**。最悪は 5711-5713 W=40 Z=2.0 で Sharpe -26.

解釈:
- 非鉄は共通因子(LME銅)で動くが、個別材料(鉱山事業・リサイクル事業の比率)で**持続的に乖離**
- 乖離は収束せず、むしろ更に拡大する傾向
- **平均回帰戦略は非鉄では使うべきではない**

## 推奨運用

### 本命戦略: 5711×5713 寄付ダイバージェンスLS

```
毎朝9:00 判定:
  5711.T ON = (寄付/前日引 - 1)
  5713.T ON = (寄付/前日引 - 1)
  差 = 5711_ON - 5713_ON

  |差| >= 1.0% なら:
    差 > 0  → 5711 Short / 5713 Long
    差 < 0  → 5711 Long / 5713 Short
  15:30引 成行で両建て決済
```

期待: **+41bps/トレード、Sharpe+3.66、週1-2トレード**

### 補助戦略: 日次10日モメンタムLS

```
毎日終値後:
  過去10営業日の累積リターン: 5711, 5706, 5713
  最強銘柄 Long / 最弱銘柄 Short
  翌日 引→引で持越し、翌々日寄で組み替え
```

期待: **+36bps/トレード、Sharpe+1.39** (補助的)

### 絶対やらない

- ❌ Zスコア平均回帰 (日次・イントラデイとも機能しない)
- ❌ 寄付ダイバージェンス 5706絡みペア (Sharpe<1.0)
- ❌ 10日以上ルックバックの逆張りランキング

## 資金効率の概算

5711×5713 寄付ダイバージェンスLS を 1銘柄500万円想定で回した場合:
- Long 500万 + Short 500万 = 建玉 1000万 (グロス)
- 1トレード期待 +41bps → 500万 × 41/10000 = **+2万円/トレード**
- 年間 83トレード × 2万円 = **+166万円/年**
- Sharpe+3.66 → 年率ボラ ~20%なら年率リターン ~73%

※Sharpe+3.66 は非常に高いが、標本(N=83)と戦争レジーム混在を考慮し **1-2単元(100-500万円)から開始推奨**
