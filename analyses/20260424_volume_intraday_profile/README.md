# 日中出来高推移 → 方向性予測

**対象**: 非鉄3 + 半導体5 = 8銘柄
**期間**: 2025-04-01 〜 2026-04-21
**アプローチ**: A. 時刻帯パネル / B. 擬似累積デルタ (OBV型)

---

## 🎯 結論: **時刻帯 vol_ratio 単独では予測できないが、「価格×累積デルタのダイバージェンス」は有効** (特に11:30時点)

| アプローチ | 最良シグナル | 結果 |
|---|---|---|
| A. 時刻帯 vol_ratio → 次30分 | 9:00-9:30 | Spearman=-0.040 (p=0.07) 実質無相関 |
| A. 時刻帯 vol_ratio → 15:30残 | 9:00-9:30 | Q1-Q5=-32 bps (p=0.22) 方向一貫性なし |
| **B. 前場終了(11:30)の 価格×累積デルタ** | **P-D+ vs P+D-** | **+28.7 bps 差, Welch t=+2.45, p=0.015** ✅ |
| B. 連続変数 diverg | 11:30時点 | r=-0.071 (p=0.008) 有意 |

## B. 擬似累積デルタ (OBV型) vs 価格 ダイバージェンス ⭐

### 定義
- `signed_vol(t) = volume(t) × sign(close(t) - open(t))`
- `cum_delta(t) = Σ signed_vol` (始値比較の意志ベース出来高)
- `z_d(t) = cum_delta(t) / total_vol(t)` (-1..+1 に正規化)
- `z_p(t) = (close(t) / open_9:00 - 1) × 10000` (bps)
- 銘柄ごとに median split して 4象限に分類

### B1. 評価時点 × 4象限 の残時間リターン

#### **11:30時点** (前場終値) → 残4時間 リターン ⭐ 最強

| 象限 | 意味 | N | mean_bps | WR | t |
|---|---|---:|---:|---:|---:|
| **P-D+** | 価格弱/デルタ強 = **強気div** | 175 | **+24.2** | 55.4% | **+2.87** |
| P+D+ | 価格強/デルタ強 = 上昇モメンタム | 536 | +3.3 | 51.9% | +0.66 |
| P-D- | 価格弱/デルタ弱 = 下落モメンタム | 530 | -1.7 | 45.7% | -0.37 |
| **P+D-** | 価格強/デルタ弱 = **弱気div** | 175 | **-4.6** | 49.7% | -0.56 |

**強気div vs 弱気div の差分 = +28.7 bps, Welch t=+2.45, p=0.015** ✅

#### 10:30時点 → 残5時間
- P-D+ (強気div): **+23.3 bps, t=+2.35**
- P+D- (弱気div): -0.9 bps, t=-0.08
- 差分: +24.2 bps (p=0.099, 有意に近い)

#### 13:30時点 → 残2時間
- P-D+: +9.1 bps / P+D-: -3.5 bps → 差+12.7 bps (p=0.11)
- 時間経過で効果減衰

#### 14:30時点 → 残1時間
- P-D+: +1.3 / P+D-: -2.7 → 差+4.0 (p=0.44)
- 実質消失

### B2. 連続変数 (price_rank - delta_rank) での Spearman

## 2. 実行可能な戦略候補: 前場終了ダイバージェンス

### 戦略「11:30ダイバージェンス」

**ルール**:
1. 11:30時点で以下を計算:
   - price_rank: 銘柄内の z_p_bps (9:00→11:30 の価格変化) の百分位
   - delta_rank: 銘柄内の z_d (cum_delta / total_vol) の百分位
2. `diverg = price_rank - delta_rank` を計算
3. `diverg <= -0.3` (強気div): 12:30寄付で **Long**
4. `diverg >= +0.3` (弱気div): 12:30寄付で **Short**
5. 15:30大引けで決済

**期待値** (集約8銘柄):
- Long側 (P-D+ 象限): mean **+24.2 bps**, t=+2.87
- Short側 (P+D- 象限): mean **+4.6 bps** (符号反転後)
- 差分: +28.7 bps
- コスト4bps後: Long +20 bps / Short +1 bps → Longサイドが実用的

**機会**: 各象限 N=175/8銘柄/年 → 1銘柄あたり年22回 × 8銘柄 = 年175回
**年間期待損益**: 175回 × 20 bps = **+3,500 bps/年** (片側)

### なぜこれが機能するのか
- **累積デルタは「機関の本音」の代理指標** — 上昇足と下落足のどちらに出来高が乗るか
- **前場までに価格と出来高の方向が乖離している状況**: 一方向に動いたが反対サイドに大きな板/約定が出ている
- 後場にかけて**実態方向 (デルタ)** に価格が収束するケースが統計的に優位

## 4. 注意事項

- **累積デルタは1分足OHLCベース** — 真のティック単位の売買圧ではなく、各分足の方向で近似
- N=175/象限 は統計限界付近。時期別ロバストネス要検証
- イラン戦争期間 (2026-02-28〜) を含む — 高ボラ環境では累積デルタの符号が増幅される可能性
- 日中VWAP戦略 (既存 eneos_vwap_oos 等) と直交するか相関するかは未検証
- スリッページは12:30寄付 → 15:30引け間にまたがるため、引け注文は一定量必要
