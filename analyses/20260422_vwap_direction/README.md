# VWAP ベース方向判定 徹底検証

**対象**: 非鉄3 + 半導体5 = 8銘柄
**期間**: 2025-04-01 〜 2026-04-20 (258営業日, 1分足)
**コスト**: 4bps (片側2bps × 往復)
**手法**: セッションVWAP (9:00起点の累積TP×V/ΣV)、6つの決定時刻 × 5つの閾値 × 2方向 (rev/trend)

---

## エグゼクティブサマリー

### 結論: **VWAP単独では厳格合格 (Sharpe>=2.0 & N>=30 & t>=2.0) ゼロ件、しかし銘柄別×時刻ではエッジあり**

| 観点 | 結果 |
|---|---|
| プール全銘柄 での相関 | 全時刻で **\|corr\| < 0.05** → 一律ルール不成立 |
| プール Reversion@th=50 最良 | 11:30で Sharpe+0.07 (残り時間短く効果薄い) |
| プール Trend@th=50 最良 | いずれも Sharpe マイナス |
| 銘柄別 Reversion 最良 | **8035 TEL: 11:30, th=120 → Sharpe+6.17, N=29, t=+2.09** |
| 銘柄別 Trend 最良 | **6857 アドバン: 10:00, th=120 → Sharpe+4.28, N=33, t=+1.55** |
| 厳格合格 (N>=30, Sharpe>=2, t>=2) | **0件** (過学習厳しく判定) |
| 緩基準合格 (N>=25, Sharpe>=2, t>=1.9) | **2件** (下記) |

**重要発見**: VWAP乖離の方向効果は **銘柄特性 (mom/rev) と時刻に強く依存**。先の first30 / ORB 分析とも整合。

## 7. 既存シグナルとの独立性 (仮説)

### VWAP Reversion は first30 シグナルと独立か?

直感: first30 は 9:30 時点、VWAP Reversion は 11:30/14:00 時点 → **時刻が違うため独立**

実際、8035 TEL の場合:
- first30 シグナル: 9:30時点で first30>+50bps → Long (Sharpe+2.54)
- VWAP Reversion: 11:30時点で VWAP+120bps超 → Short (Sharpe+6.17)

これらは相関低く、**重ね掛けで期待値増**。例えば:
- 朝は first30 シグナルでLong建て、11:30でVWAP上離れていれば利確&ドテン、みたいな運用

### 4063 信越化学は Trend が各所で一貫

- first30: >+50bps Long (Sharpe+6.04)
- VWAP 10:00: dev>=50bps Long (Sharpe+2.95)
- VWAP 11:30: dev>=80bps Long (Sharpe+3.46)

→ **モメンタム性格が時刻を通じて安定**、追加エントリー or サイズ増の根拠に使える。

## 8. VWAP 戦略の限界と注意

### 限界
1. **プール (一律ルール) ではゼロエッジ** — 銘柄選別必須
2. **N が小さい** (th=120では 20-50/年) → 過学習リスク
3. **閾値感応度が強い** — 80/120/50 で Sharpe が大きく変わる
4. **14:00 など遅い時刻の戦略は残り時間 1.5h** → コスト負けリスク

### 注意
- VWAP は**累積計算**なので、前場中盤では初期値支配で変動小
- 流動性低い分足 (volume=0) があると VWAP が ffill され**乖離測定誤差**
- 指数系銘柄 (本分析では該当なし) では volume=NULL で計算不可
- **リーク確認済み**: 決定時刻 T の close のみ使用、T+1min以降は未使用

## 11. 次の検証候補

1. **VWAP Slope** (傾き): 直近5/15分のVWAP上昇率を追加特徴量に
2. **二重決定**: 11:30 reversion + 14:00 追加エントリー のコンボ
3. **VWAP + Volume**: OR volume ratio と VWAP dev の AND/OR 条件
4. **Dynamic threshold**: 閾値を **当日のATR** でスケーリング
5. **Walk-forward OOS**: 6か月学習 → 3か月OOS で過学習チェック
