## 実運用アイデア (セクター特化)

### A. Post期間で機能する可能性の高い戦略

1. **★★ 水曜 半導体Long (引け跨ぎ)**: 全日+332bps/日 (t=+3.85)
   - 火曜引け買い → 水曜引け決済、もしくは水曜寄→水曜引
   - バスケット(5銘柄等加重)で実行、1日で大きなエッジ

2. **★★ 水曜 半導体Long + 非鉄Short ペア**: 日中スプレッド+199bps
   - 水曜9:00寄: 半導体Long / 非鉄Short
   - 15:30引: 決済
   - 銅価格変動の影響を消しながら「半導体の水曜日中優位」だけ抽出

3. **★ 月曜 三菱マテリアルShort** (-79bps, t=-2.50)
   - 月曜9:00寄Short → 15:30引決済
   - 非鉄の中でも最も月曜が弱い銘柄

4. **★ 金曜 ディスコLong** (+94bps, t=+3.51)
   - ただしN=4なので1-2週で再検証

### B. 新本命候補: 水曜半導体バスケットLong

半導体5銘柄の中で:
- 水曜Long個別: TEL+96, アドバン+123, ディスコ+11, 信越+36, ローム+106
- **3銘柄 (TEL/アドバン/ローム)** が突出
- **推奨構成: 水曜 TEL + アドバン + ローム の3銘柄等加重Long**
- 期待 ~+108bps/日 (平均)、N=3〜5 で標本は薄いが継続観察の価値大

### C. 避けるべき取引

- ❌ 非鉄の水曜/木曜 寄付でLong (寄り天でON+378→日中-123〜-147吐く)
- ❌ War期間に5713 住友金属鉱山をLong持越し (-25% DDの実績)
- ❌ 火曜 寄りで半導体/非鉄のLong (-107〜-78bps/日)

### D. LME銅本命戦略への修正提案

旧「LME銅+1% → コア5Long」は戦後 月/木 でドラッグ。
非鉄・半導体に特化すると:
- **半導体5銘柄Long版**: 水曜実行でエッジ極大、月曜ON除外
- **非鉄3銘柄Long版**: 寄付ではなく前日引けから保有 (ONのエッジを取りに行く)
- 非鉄は**日中吐くので「引け→翌寄り」(ON)保有だけに絞る** のが合理的
