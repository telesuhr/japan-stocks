## 超シンプル版 (3取引, 期待+788bps/週)


## 水曜ペア版 (追加オプション, +199bpsスプレッド)

水曜日中のセクター乖離を取りに行くペアトレード。
非鉄の寄り天を利用 → 寄り非鉄Short / 半導体Longの相対価値。

```
火引                       水寄                     水引
 │                          │                       │
[非鉄Long 新規]─────────→ [非鉄 寄り決済]
[半導体Long 新規]───────────────────────────→ [半導体 引け決済]
                           [非鉄Short 新規]───────→ [非鉄Short 引け決済]
                           (半導体LongはそのままHold)
```

- 火引 : 非鉄Long + 半導体Long 両方新規
- 水寄 : 非鉄Longを利確 (+378bps取る) → 同時に非鉄Shortを新規建て
- 水引 : 半導体Long + 非鉄Short を同時決済
  - 半導体Long日中: +76bps
  - 非鉄Short日中: +123bps (-123bpsの逆)
  - **ペアスプレッド +199bps**

### 毎週のルーチン

| 曜日 | 時刻 | アクション |
|------|------|-----------|
| **金** | 15:25-30 | 半導体3銘柄 (TEL/アドバン/ローム) 引成Short |
| **月** | 9:00 | 半導体Short 寄成買戻し (決済) |

| **火** | 15:25-30 | 半導体Short 引成買戻し → **同時に** 半導体5銘柄 + 非鉄3銘柄 を引成Long |
| **水** | 9:00 | 非鉄3銘柄 寄成売り (決済) ※半導体はホールド |
| **水** | 15:25-30 | 半導体5銘柄 引成売り → **同時に** 非鉄3銘柄 + 半導体5銘柄 を引成Long |
| **木** | 9:00 | 非鉄 + 半導体 両方 寄成売り (決済、ONの反発取り切り) |
| **金** | 終日 | ノーポジ or 次週の金引Shortに備える |

## 資金所要額の目安

バスケット等加重で1銘柄100万円想定:
- 非鉄3銘柄 = 300万円
- 半導体5銘柄 = 500万円
- **最大建玉: 火引〜水引 で 非鉄300 + 半導体500 = 800万円Long + 水寄で非鉄300Shortクロスする場合は 1,100万円**
- 信用口座なら同額の証拠金で可

## 運用上の注意

- **シンプル版3取引からスタート推奨**。フル版は7取引でコスト・発注ミスのリスク上昇
- **水曜ONのエッジは最も頑健** (半導体t=+3.22) → これを軸に据える
- 週次で実績を記録、**4週連続で想定下回ったら一旦停止 → 再検証**
- SQ週(3月第2金曜・6月第2金曜等)、FOMC/日銀会合週 は曜日効果が壊れる → 除外
- 6963 ローム は War期間逆行高 +1538bps、**半導体バスケットの主力に据えて安全マージン確保**
- 5713 住友金属鉱山 は War-25%DD、**非鉄バスケットから外す or サイズを半分にする**検討
