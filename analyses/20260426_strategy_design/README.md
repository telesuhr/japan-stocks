### タイミング問題の整理

```
LME木曜シグナルの時系列:
  Wed 終値確定 (JST 3:00 頃)     → シグナル確定
  Wed 市場は既に引け済み          → 前日引けエントリー不可
  Thu 9:00 寄付                  → ONリターンは既に発生済み
  Thu 15:30 引け                 → V2(Intra)は ON織込後の後追い
```

**V1 (ON) の問題**: シグナル確定 (JST 3:00) から寄付 (9:00) の間に入る必要がある。
  - 夜間 PTS / 翌朝の指値待機が必要
  - 理論 Sharpe+28.54 は理想値。実際は寄値が高くなりエントリーコスト増

**V2 (Intra) の問題**: 寄付後エントリーのため ON 分は取り込めない。
  - 実現 Sharpe+2.60 (V1の理論値より大幅劣後)
  - しかし操作が単純で月曜海運Short と同等の実装難度

### 採用判定

**V2 (Intra) で近似実装**。
- 9:00 寄付エントリーはマニュアル発注で十分対応可能
- 月1-2回の低頻度シグナル → 集中管理しやすい
- Sharpe+2.60 は閾値 (+2.0) を超えており採用ライン達成
- V1 (ON前日引けエントリー) は将来的に夜間注文が使えれば検討

## 限界・注意点

1. **S1 N=45**: t=+2.17 は採用ライン最低限。負け月3ヶ月が特定レジームに集中していないか継続観察
2. **S2 N=10**: サンプルが非常に薄い。V2 Sharpe+2.60 は N=10 時点の推定値。次の5回で大きく動く可能性
3. **信用コスト**: S1 ショートは信用金利 (年1-2%) を加算して採算評価すること
4. **流動性**: 海運3銘柄は流動性高いが、エネルギー中小 (5016) はスプレッド広い日がある
5. **レジーム変化**: Post (戦後) は N少なく将来のデータで要再評価
