# A+B 合成戦略 — 市場レジーム × 個別銘柄シグナル

## 結論を先に

✅ **「A強気レジーム × B空売り急増銘柄ロング」で Sharpe が B単独より 60% 改善** (0.08 → 0.13)。  
✅ **「LS両建て」** (A強気→Long / A弱気→Short) で Sharpe 0.14 + t=3.50 → 統計的に強い。  
ただし**個別株分散が大きく、Sharpe自体は低水準**にとどまる。実用には銘柄絞り込みが必要。

---

## 戦略アイデア

| 仕組み | 何を判定するか |
|---|---|
| **Phase A (週次)** | 市場全体のレジーム (海外/投信/個人フロー) で「強気週」「弱気週」を判定 |
| **Phase B (個別銘柄)** | 銘柄ごとの空売り建玉変化で個別シグナル抽出 |

組み合わせ:
- A強気 → 個別銘柄で空売り急増 = **squeeze候補 + 市場tailwind** → ロング
- A弱気 → 個別銘柄でショートカバー = **機関カバー失敗 + 市場逆風** → ショート

## 主要発見

### 1. **Aレジームフィルタの効果は確実だが小さい**

| 戦略 | mean | Sharpe |
|---|---|---|
| B単独 (空売り急増→Long) | +11.9 | 0.08 |
| A強気 × B空売り急増 (合成) | **+18.5** | **0.13** |
| 改善幅 | +55% | +63% |

つまり**「市場が強気な週だけB戦略を実行」することで、約60%パフォーマンス改善**。  
ただし絶対値は依然小さい (mean +18bps/5d, Sharpe 0.13)。

### 2. **A単独 (市場ニュートラル化済) は当然ゼロ**

「A強気週の全銘柄ロング」だと市場ニュートラル化後リターンは 0.0bps。  
当然: 「市場全体が強気」と「個別銘柄が市場アウトパフォーム」は別の話。  
**A は方向性 (TOPIX上下)、B は銘柄選別 (相対パフォーマンス)** という別軸の情報。

### 3. **LS両建ては最も実装しやすい**

「A強気→ロング, A弱気→ショート」のシンプルなルール:
- Sharpe 0.14
- t=3.50 (統計的に強い)
- 勝率 48.9% (平均的)
- mean +18.7bps/5d

### 4. **B単独の「ショートカバー逆張りショート」が依然最強**

| 戦略 | Sharpe |
|---|---|
| **B単独: 空売りカバー Top10% → Short** | **0.17** ← 最強 |
| LS両建て (A+B) | 0.14 |
| A強気 × B空売り増 Long | 0.13 |

Aフィルタ不要で、Bだけで Sharpe 0.17 達成。  
Aを加える価値は「絶対リターン底上げ」というよりは「**取引頻度を下げてリスク管理**」。

---

## 解釈

### 「Aフィルタで頻度を絞る」 意義

| 戦略 | N (5年で) | 年あたり取引週 |
|---|---|---|
| B単独 Long | 58,051 | 1,140週分の銘柄 |
| A+B Long | 20,673 | 410週分の銘柄 (1/3) |

Aフィルタで取引頻度が1/3になる → **コスト負担が1/3に縮小**。  
これは Sharpe には反映されていないが、実運用では大きな差になる。

### 個別株シャープが低い理由

個別株のリターン std = 978-1045 bps と非常に大きい。  
mean +18-22bps では Sharpe 0.1-0.2 にしかならない。

**解決策**:
1. 銘柄絞り込み: 上位 30銘柄に集中
2. リスクパリティ: 高ボラ銘柄のウェイト下げる
3. 保有期間延長: 5日 → 15日でリターン拡大 (std はその分大きくならない)

```
B単独 累積:    +13,889 bps  (5年)

```

絶対値では B単独が最大だが、これは銘柄数が多いから。  
**1銘柄あたり** で見れば A+B 合成の方が効率的。


## 限界・注意点

1. **Sharpe絶対値が低い**: 0.13-0.17 ではコスト後にエッジが残らない可能性
2. **個別株分散が大きい**: std 1000bps → 1銘柄数百万円ロットでは結果のブレが激しい
3. **コスト含めて評価**: 個別銘柄売買は片側3-5bps → 往復6-10bps → Sharpe半減
4. **データ更新頻度の差**: A=週次, B=日次 → タイミングずれ
5. **A弱気の継続性**: Aが弱気と判定した週でも、翌週いきなり強気に転じる可能性
