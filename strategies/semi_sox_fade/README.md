# semi_sox_fade — 半導体 SOX フェード戦略

## 概要
SOX 指数 (Philadelphia 半導体指数) の前日 NY セッション変化が ±30 bps 超のとき、
**逆方向**に日本半導体 5 銘柄を寄付エントリ → 大引け決済する戦略。

仮説: SOX 強い夜は日本半導体株が GU で寄付くが、寄付後は織り込みが行き過ぎて
イントラデイで反落する傾向 (corr(SOX, gap)=+0.65, corr(SOX, open→close)=-0.10)。

## シグナル仕様

### 計算
- 直近 NY セッション (asof より前で最新の SOX trade_date) の close 前日比 (bps)

### エントリ
- `|SOX 変化| ≥ 30 bps` → 5 銘柄を寄付で **逆方向** に発注
- SOX > 0 → SHORT
- SOX < 0 → LONG

### エグジット
- **大引け (15:30)** で全銘柄寄成反対売買

## ポジションサイジング
- 推奨総資金: ¥1,500-2,500 万
- 各銘柄: 総資金の **1/5** を均等配分
- レバ最大 2 倍

## 他戦略との関係
- 既存 `sox_overnight_short` (Sharpe 2.11) と方向性は同じだが、本戦略は **5 銘柄分散 + 寄付エントリ + 大引け決済** で構成が異なる
- `nonferrous_lme_link` とは独立 (異セクター)
- `pair_portfolio` (Sharpe 1.37) とも低相関期待
