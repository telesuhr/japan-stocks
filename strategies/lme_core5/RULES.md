# 意思決定フローチャート (v1.1 / ONホールド版)

⚠️ **重要**: 本戦略は **Day N 大引け引成Long → Day N+1 寄付寄成決済** (ONホールド ~17.5時間)。
シグナル = LME銅 東京セッション中 (JST 9:00 → 15:25) の変化率。

---

## 🟢 Day N 15:15 — トレードするか?

```
START (Day N 15:15)
  │
  ▼
① LME銅 東京セッション変化率 (JST 9:00 → 15:25) が +1.0% 以上か?
  │
  ├── No  → スキップ (skipped_reason=lme_below_threshold)
  │
  └── Yes ▼

② 本日 (エントリー日 Day N) は木曜日か?
  │
  ├── Yes → スキップ (skipped_reason=thursday)
  │         ※ 木曜エントリー→金曜寄付決済は Sharpe -1.31
  │
  └── No  ▼

③ Day N または Day N+1 に FOMC/米雇用統計/日銀会合が控えているか?
  │
  ├── Yes → サイズ半減 (各¥500万, 合計¥2,500万) or スキップ
  │
  └── No  ▼

④ 5銘柄のうち、Day N または Day N+1 が決算発表日の銘柄があるか?
  │
  ├── Yes → その銘柄のみ除外、他銘柄で実行
  │         除外後の銘柄数 < 3 なら全体スキップ
  │
  └── No  ▼

⑤ Day N+1 が配当落ち日の銘柄があるか?
  │
  ├── Yes → その銘柄のみ除外 (配当分を価格で調整する余裕がないため)
  │
  └── No  ▼

⑥ 日経225先物・CMEが Day N 15:20 時点で急落中 (-2%超) か?
  │
  ├── Yes → スキップ (skipped_reason=panic)
  │
  └── No  ▼

✅ Day N 15:27-15:29: 5銘柄 引成 Long 発注
   Day N 15:30 大引け: 約定
```

## 🌙 Day N 夜間 — ONホールド中

```
⑦ ロンドンセッションで LME銅 -3% 超急落しているか?
   (JST 22:00頃時点)
  │
  ├── Yes → Day N+1 寄付前に部分損切り検討 (任意, デフォルトは保有継続)
  │
  └── No  ▼

保有継続 → Day N+1 朝へ
```

  │

  │

   Day N+1 09:00 寄付: 約定 → P&L確定

## 📊 ポジションサイジング ルール

```
基準: 1銘柄 = ¥1,000万, バスケット合計 = ¥5,000万

特殊状況:
IF 対象銘柄数 = 5:        各¥1,000万 (標準)
IF 対象銘柄数 = 4 (1銘柄除外): 各¥1,000万 (合計¥4,000万、再配分しない)
IF 対象銘柄数 = 3:        各¥1,000万 (合計¥3,000万)
IF 対象銘柄数 < 3:        全体スキップ
IF マクロイベント前夜:     各¥500万に半減
IF 直近3ヶ月連続負け:      各¥500万に半減 (暫定)
IF 年間DDが -10% 超:      年内撤退
```

## 🔒 リスク上限

```
1トレード: バスケット Gross P&L -2% = -¥100万 超なら翌寄付で即決済
月次:      月間 Net P&L -¥200万 到達で月内新規エントリー停止
年次:      年間 Net P&L -¥500万 到達で戦略停止・再検証
```

ONホールド特有のテールリスク:

| ケース | 想定頻度 | 想定損失 |
|---|:-:|:-:|
| 通常 ON 変動 | 常時 | ±0.5〜1.5% |
| ネガティブサプライズ決算 | 四半期1回/銘柄 | -3〜5% |
| CME・NY急落 (-3%超) 翌寄付 | 年数回 | -2〜4% |
| 地政学イベント夜間発生 | 年1-2回 | -3〜7% |
| ブラックスワン (戦争・金融危機) | 数年に1回 | -5〜10%以上 |

## 🔁 継続運用の判断基準

月次レビューで確認:

| 指標 | 良好 | 注意 | 撤退 |
|---|---|---|---|
| 勝率 (直近3ヶ月) | ≥60% | 50-60% | <50% |
| 平均リターン/トレード | ≥+100bps | +50〜100 | <+50 |
| Sharpe (直近12トレード) | ≥+5 | +3〜5 | <+3 |
| 実現 MaxDD | <-3% | -3〜-5% | <-5% |
| ON1回あたり最大損失 | <-2% | -2〜-4% | <-4% |

**「撤退」列が2つ以上該当 → 戦略一時停止**

## 💡 裁量介入ルール

| 状況 | 許容される裁量 |
|---|---|
| 大規模地政学イベント発生中 | 当日スキップ |
| FOMC/日銀当日・翌日 | スキップ or サイズ半減 |
| 個別銘柄の重大ニュース | その銘柄のみ除外 |
| ロンドン時間でLME -3%超 | 翌朝の部分損切り |
| 「嫌な予感」 | 年3回以内で許容 |

年間裁量スキップ回数 > 5回 → ルール厳守に戻す。

## 📝 記録必須事項

- シグナル発動有無
- LME 9:00 JST価格, 15:25 JST価格, 東京変化率
- エントリー曜日 / 決済曜日
- 各銘柄 entry/exit 価格・株数・bps
- 手数料 (片道2bps × 往復)
- Gross / Net P&L
- ONホールド中の市場イベントメモ
- スキップ時は skipped_reason 必須

記録なきトレードはバックテスト検証で使えない。
