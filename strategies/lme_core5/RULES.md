# 意思決定フローチャート (Yes/No)

毎朝、以下のフローを上から順に辿る。各分岐で「いいえ」に落ちたら**スキップ**。

---

## 🟢 本日トレードするか?

```
START
  │
  ▼
① 前日 LME銅終値が前々日比 +1.0% 以上か?
  │
  ├── No  → スキップ (skipped_reason=lme_below_threshold)
  │
  └── Yes ▼
  
② 本日は木曜日か?
  │
  ├── Yes → スキップ (skipped_reason=thursday)
  │
  └── No  ▼

③ 日経平均の寄付が前日比 -2% 以上のギャップダウンか?
  │
  ├── Yes → スキップ (skipped_reason=panic_gap)
  │
  └── No  ▼

④ 5銘柄のうち、本日±1営業日が決算発表日の銘柄があるか?
  │
  ├── Yes → その銘柄のみ除外、他4銘柄で実行
  │
  └── No  ▼

⑤ 証券口座・ネット接続・取引時間に問題ないか?
  │
  ├── No  → スキップ (skipped_reason=system_failure)
  │
  └── Yes ▼

✅ 5銘柄 (または該当除外後の銘柄) で 9:00 寄成 Long 発注

  │
  ▼
保有 (9:00 - 15:25)
  │
  ▼

⑥ 個別銘柄が -5% 以上急落、または日経が 11:00 時点 -3% 超か?
  │
  ├── Yes → 該当銘柄のみ手動クローズ (or 全決済)
  │
  └── No  ▼

  保有継続
  │
  ▼
  
✅ 15:25 に引成 売り発注
✅ 15:30 決済 → P&L 記録
END
```

## 📊 ポジションサイジング ルール

```
基準: 1銘柄 = ¥1,000万, バスケット合計 = ¥5,000万

特殊状況の対応:

IF 対象銘柄数 = 5: 各¥1,000万 (標準)
IF 対象銘柄数 = 4 (決算1銘柄除外): 各¥1,000万 (4,000万 のまま、再配分しない)
IF 対象銘柄数 = 3 以下: トレード全体を見送り
IF 直近3ヶ月連続で負け: 各銘柄を¥500万に半減 (暫定的)
IF 年間DDが -10% 超: 年内撤退
```

---

## 🔒 リスク上限

```
日次:  1トレードあたり想定損失 -2% = -¥100万 を超えた場合は手動クローズ
月次:  月間P&L が -¥200万 (累積) に達したら月内の新規エントリー停止
年次:  年間P&L が -¥500万 に達したら戦略停止し、再検証
```

---

## 🔁 継続運用の判断基準

月次レビュー (毎月初日) で以下を確認:

| 指標 | 良好 | 注意 | 撤退 |
|---|---|---|---|
| 勝率 (直近3ヶ月) | ≥60% | 50-60% | <50% |
| 平均リターン/トレード | ≥+100bps | +50〜100 | <+50 |
| Sharpe (直近12トレード) | ≥+5 | +3〜5 | <+3 |
| 実現 MaxDD | <-3% | -3〜-5% | <-5% |
| LME銅 月次変動 | 通常 | -5〜-10% | <-10% |

**「撤退」列が2つ以上該当 → 戦略一時停止、原因分析後に再開判断**

---

## 💡 裁量介入ルール

**原則: 裁量介入は最小限**。しかし以下の場合は許容:

| 状況 | 許容される裁量 |
|---|---|
| 大規模地政学リスク (戦争・テロ) の直後 | 当日スキップ |
| 日銀政策会合・FOMC 当日 | 当日スキップ or サイズ半減 |
| 個別銘柄に重大ニュース (不祥事・リコール等) | その銘柄のみ除外 |
| 「何となく嫌な予感」 | スキップ可 (年3回以内) |

年間の裁量スキップ回数が 5回を超える → 裁量介入を制限 (ルール厳守に戻す)

## 📝 記録必須事項

以下は必ず `trade_log.csv` に記録:

- シグナル発動有無 (Yes/No)
- スキップ理由 (該当時)
- LME銅変化率
- 曜日
- 各銘柄のエントリー・決済価格
- 各銘柄の bps リターン
- 手数料
- ネットP&L
- 特記事項 (notes)

記録なきトレードは「なかったこと」扱い。後続のパフォーマンス検証で使えない。
