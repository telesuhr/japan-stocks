# 📋 日次オペレーションチェックリスト (v1.1 / ONホールド版)

⚠️ **重要**: 本戦略は **当日大引け引成エントリー → 翌営業日寄付寄成決済** の ONホールド戦略。
シグナルは「LME銅 東京セッション中 (JST 9:00 → 15:25) の変化率」で判定する。

所要時間: Day N 15:15-15:35 に約15分 + Day N+1 08:45-09:05 に約10分。

---

## 🕒 Day N 15:15 - 15:25 — シグナル判定

### 1. LME銅 東京セッション中の変化率を確認

- [ ] Reuters/Bloomberg/investing.com で **CMCU3** の以下2点を記録:
  - JST 9:00 時点の価格 (東京寄付時)
  - JST 15:25 時点の直近値 (東京引け前)
- [ ] 変化率を計算: `(15:25価格 / 9:00価格 - 1) × 100`

```
JST 9:00  LME銅: $______________
JST 15:25 LME銅: $______________
東京セッション変化率: ______________%
```

**変化率 < +1.0% なら本日はスキップ → skipped_reason=lme_below_threshold**

補助スクリプト:
```bash
cd ~/claude-code/japan-stocks/strategies/lme_core5
```

## 🚫 スキップ理由コード

- `lme_below_threshold`: 東京セッション変化率 < +1.0%
- `thursday`: 木曜エントリー除外
- `earnings`: Day N または Day N+1 が決算日
- `macro_event`: FOMC/雇用統計前 (サイズ半減で代替も可)
- `panic`: 日経先物夜間が急落中
- `system_failure`: 証券会社システム障害
- `discretionary`: 手動裁量スキップ
