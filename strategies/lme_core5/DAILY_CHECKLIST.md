# 📋 日次オペレーションチェックリスト (印刷用)

毎朝、以下の順序でチェックする。所要 約10分。

---

## 🌅 朝のシグナル確認 (06:00 - 08:30)

### 1. LME銅終値の確認

- [ ] Reuters/Bloomberg/investing.com で **CMCU3** (LME Copper 3-Month) の前日終値を確認
- [ ] 前々日終値からの変化率を計算: `(前日終値 / 前々日終値 - 1) × 100`
- [ ] **変化率が +1.0% 以上** か確認

```
前々日 LME終値: $______________
前日  LME終値: $______________
変化率:        ______________%
```

**+1.0% 未満なら本日はスキップ → チェックリスト終了**

---

### 2. 曜日チェック

- [ ] 本日の曜日は ?
  - 月 / 火 / 水 / 金 → OK
  - **木曜 → 本日はスキップ** (曜日アノマリー)

---

### 3. 決算発表カレンダー確認

以下の5銘柄について、**本日±1営業日が決算発表予定日でないか** を確認:

- [ ] 5711 三菱マテリアル ... _______ (決算予定?)
- [ ] 6501 日立製作所 ........ _______
- [ ] 7011 三菱重工業 ........ _______
- [ ] 5016 出光興産 .......... _______
- [ ] 4502 武田薬品 .......... _______

**該当銘柄がある場合、その銘柄のみ除外して残り4銘柄で実行。**

参考: 企業HP or 株探 (https://kabutan.jp/) → 銘柄検索 → 決算発表日

---

### 4. 特殊状況チェック

- [ ] 日経平均の前日寄付が -2% 以上のパニックギャップでない
- [ ] 対象銘柄にストップ安・売買停止がない
- [ ] 配当落ち日でない (該当なら別計算)
- [ ] 大規模地政学イベント直後 (戦争勃発1週間以内) でない

**すべて OK なら → 発注フェーズへ**
