## lme_on_copper との関係 / ポートフォリオ運用

両戦略は**同じ銘柄バスケット・同じエントリー/決済タイミング**だが、**シグナル源と判定時刻が異なる**:

| 項目 | lme_on_copper | topix_overnight |
|---|---|---|
| シグナル源 | **当日** LME銅 東京時間変化率 | **前日終値→当日9:00** TOPIX夜間ギャップ |
| 判定時刻 | Day N **15:25** (引け直前) | Day N **09:05** (朝) |
| 発動頻度 | 月1-2回 | 月4-5回 |
| Sharpe | **+11.50** | +4.79 |

**構造の違い**:
- `lme_on_copper`: 当日の日中LME動向を終日監視し、引け直前に判断
- `topix_overnight`: 朝9:05に判断確定。残り6.5時間は通常業務しながら待つだけ

### 併用ルール

- **両シグナル同時発動**: ポジションは**1回分のみ** (¥5,000万)。重複してダブルベットしない
- **片方のみ発動**: 発動した戦略をそのまま実行
- **優先順位**: lme_on_copper 発動日は lme_on_copper を優先 (Sharpeが高い)

## オペレーション概要

### Day N 08:55 — TOPIX朝シグナル判定
1. TOPIX (.TOPX) の前日終値を記録
2. 09:00直後 (09:00-09:05) の TOPIX 値を確認
3. 変化率 = (09:00値 / 前日終値 - 1) × 100

### Day N 09:05 — 判定
- ≥ +0.3% → 本日 15:30 引成エントリー予定
- < +0.3% → スキップ

### Day N 15:27-15:29 — 発注
CORE5 各 ¥1,000万 を 引成 (CLO) で Long

### Day N+1 08:55-08:59 — 決済発注
5銘柄 寄成 (OPG) で Sell

詳細は `DAILY_CHECKLIST.md` / `RULES.md` を参照。
