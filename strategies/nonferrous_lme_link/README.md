# nonferrous_lme_link — 非鉄株 LME 銅オーバーナイト連動戦略

## 概要
LME 銅 (CMCU3) の **JST 朝までの 24 時間変化率** が ±一定 bps 超のとき、
日本市場の非鉄・電線 8 銘柄を寄付で同方向にエントリし、午前中ないし大引けで決済。

## シグナル仕様

### 計算
- **JST 当日 08:55 直近の CMCU3 価格** vs **24h 前の同価格** → bps 変化
- 12時間以内に LME データが存在しない場合はシグナル無効

### エントリ
1. `|ON 変化| ≥ 150 bps` → TIER1
   - 方向: ON > 0 なら 8 銘柄を Long、ON < 0 なら Short
   - 寄成 (09:00) でエントリ → 大引け (15:30) で反対売買
2. `80 ≤ |ON 変化| < 150 bps` → TIER2
   - 同方向で寄成エントリ → 11:00 で反対売買

### 同日重複時
- TIER1 ルール優先 (大引けまで保有)

## ポジションサイジング
- 推奨総資金: ¥2,000-3,000 万
- 各銘柄: 総資金の **1/8** を均等配分
- レバ最大 2 倍まで (MDD 想定 384bps × 2 = 768bps ≈ 8% 想定)

### 個別ウェイト調整 (任意)

- 5802 を 0.5 倍ロット
- 上記調整で全体 Sharpe を更に +0.3-0.5 改善見込み
