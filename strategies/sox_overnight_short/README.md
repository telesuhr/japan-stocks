- SGX は JGB未扱い、TOPIX mini は大阪

## `topix_overnight` (Long) との関係

両戦略はシグナル方向が**逆**:

| 項目 | topix_overnight (Long) | sox_overnight_short (Short) |
|---|---|---|
| シグナル | TOPIX ギャップ ≥ +0.3% | .SOX ≤ -2% |
| 判定 | Day N 09:05 | Day N-1 米国引け後 (翌朝再確認) |
| エントリー | Day N 15:30 引成 Long | Day N 09:00 寄成 Short |
| 決済 | Day N+1 09:00 寄成 | Day N 15:30 引成 |
| 方向 | Long (バイアス+) | **Short (バイアス-)** |
| バスケット | CORE5 5銘柄 | 1306.T 単独 |
| ONリスク | あり | **なし (日中のみ)** |

### 同日発動時のルール

| 状況 | 対応 |
|---|---|
| SOX Short のみ | SOX Short 実行 |
| topix_overnight のみ | topix_overnight 実行 |
| **両方発動** | **SOX Short を優先** (当日売り先行 → Long は見送り) |
| lme_on_copper と SOX Short 両発動 | 両方実行可 (時間帯・方向・商品が独立) |

**理由**: .SOX < -2% = 米半導体急落 → 当日の日本市場は下落バイアス。この状況で TOPIX Long (topix_overnight) を仕掛けるのは逆行。
