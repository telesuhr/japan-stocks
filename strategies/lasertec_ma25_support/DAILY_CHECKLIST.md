# 毎日のオペレーション — lasertec_ma25_support

## 🕓 15:30 (引け後) — エントリー判定

`signal_check.py` を実行、または手動で以下を確認:

1. [ ] 本日 6920.T の終値を取得
2. [ ] 直近20営業日の高値を計算 → dd20 = 終値/高値-1
3. [ ] MA25 (過去25日終値平均) を計算
4. [ ] MA25 5営業日前と比較 (slope 判定)
5. [ ] 本日 日中High/Low と MA25 ±1.0% 比較 (touch 判定)

**全て YES ならエントリー候補確定**:
- dd20 ≤ -5%
- MA25 ±1% に接触
- MA25 が 5営業日前より高い

除外チェック:
- [ ] 現在 6920.T のポジションあり? → スキップ
- [ ] 明日 FOMC/日銀会合/メジャーSQ? → スキップ
- [ ] 6920.T 決算発表が3営業日以内? → スキップ
- [ ] 6920.T 売買停止/信用規制? → スキップ

## 🕘 翌営業日 09:00 — 寄成 Long 実行

- [ ] 寄付前 (08:59) に 6920.T 成行Long注文 (サイズ ¥500-1,000万)
- [ ] 09:00 約定確認、約定価格 P_entry を記録
- [ ] 即座に逆指値 (成行) **P_entry × 0.90** を発注 (Stop)
- [ ] `trade_log.csv` に記録 (entry_date / entry_price / stop_level / planned_exit_date)

## 🕒 建玉中 (1〜10営業日)

- [ ] 毎日 15:30 前に Stop が有効か確認
- [ ] Stop ヒット時: 成行約定確認 → `stop_hit=True` / `exit_price` / `exit_date` を記録
- [ ] 地合い急変 (FOMC急変、地政学リスク大) → 臨時判断で手仕舞い検討

## 🕓 10営業日後 15:25 — 時間決済

- [ ] Stop がヒットしていなければ、10営業日後の 15:25 引成で決済
- [ ] 約定確認、`exit_date / exit_price / net_ret_pct` を記録

## 🌙 取引終了後

- [ ] 本日のP&L (保有分含む MTM)
- [ ] 月次: 発動回数、WR、平均リターン、想定 Sharpe+7.68 との乖離

---

## 🚨 異常時対応

| 状況 | 対応 |
|---|---|
| 寄付が特別気配継続 | 寄付不可 → 当日スキップ、翌営業日に再判定 |
| Stop 逆指値が板で消滅 (約定後に再発注漏れ) | 即座に再発注 |
| レーザーテック決算サプライズで建玉中に急落 | Stop に任せる、余裕あれば手動早期Exit |
| サーキットブレーカー | 建玉を成行フラット化 |
| 大口主要顧客 (ASML等) のネガティブニュース | 当日フラット化を検討 |
