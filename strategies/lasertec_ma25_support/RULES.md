## 詳細ルール

### 判定タイミング
- 判定は引け後 (15:30 以降) に行う (MA25, dd20, 上昇判定すべて当日終値ベース)
- 実行は翌営業日 寄成

### MA25 の計算
- 過去 **25営業日** の終値平均 (当日終値を含む)
- 毎日更新、翌営業日のエントリー判定に使う

### 接触判定
```
触れた = (当日Low ≤ MA25 × 1.01) AND (当日High ≥ MA25 × 0.99)
```
日中のどこかで MA25 ±1.0% 以内に入ったら接触と見なす。

### 上昇判定
```
上昇中 = (当日MA25 > 5営業日前のMA25)
```
4営業日前との比較ではなく、**5営業日前** と比較 (傾きとして明確にするため)。

### Stop の約定想定
- 板に **逆指値成行** を置く (P_entry × 0.90 到達で成行発注)
- 実際の約定は stop level から 1-2% 下で滑る可能性 (成行slippage)
- バックテストでは stop level で fill と仮定

### 時間決済
- 10営業日後 = エントリー日の10営業日先
- 例: 月曜エントリー → 翌週金曜 引成決済 (間に休日なしの場合)
- 引成決済が約定不能 (特別気配等) の場合は翌営業日寄成

### 同銘柄のポジション重複
- 原則として1ポジションのみ (先行ポジションの決済完了まで新規スキップ)
- ただしバックテスト上は重複を許容していたため、実運用初期は同時2ポジションまで許容して様子見

## 除外・スキップ条件
- [ ] 保有中ポジションあり
- [ ] 翌日が FOMC / 日銀会合 / メジャーSQ
- [ ] レーザーテック決算発表日が翌日 ~ 3営業日以内
- [ ] レーザーテック 売買停止 / 信用規制
- [ ] 直近3ヶ月の月次戦略リターンが連続マイナス → 一時停止中
