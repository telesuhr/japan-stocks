# lasertec_ma25_support — レーザーテック MA25 サポート狙撃

## 戦略概要
レーザーテック (6920.T) の日足で、**下落局面 (20日高値から-5%以上) に日中価格が MA25 に接触し、MA25 自体が上昇中** の場合に翌営業日寄成で Long。10営業日後 引成決済、またはエントリー価格から -10% で損切り。

### パラメータ頑健性
グリッド (ma 15-50, tol 0.5-2.0%, dd 3-10%) の上位20設定すべて t > 5、mean +4〜+7%。オーバーフィットの兆候なし。

## エントリー/エグジット ルール

### エントリー条件 (すべて満たす日の**終値後**に確定 → 翌営業日寄成で Long)
1. **下落局面**: 当日終値が 直近20営業日高値から -5% 以下
2. **MA25 接触**: 当日 日中Low が MA25 × 0.99 ~ 日中High が MA25 × 1.01 の範囲 (= MA25 に±1.0%以内で触れた)
3. **MA25 上昇中**: 当日 MA25 > 5営業日前の MA25
4. **同日重複**: 前日までに保有中のトレードがあれば新規スキップ (単銘柄で積み上げない)

### エグジット
- **時間決済**: エントリー日から **10営業日後** の終値で引成決済
- **ストップロス**: エントリー後、日中Lowがエントリー価格の **-10%** 到達 → 成行決済 (Stop)

### サイズ
- 1トレードあたり ¥500-1,000万
- 銘柄集中のため、ポートフォリオ全体の5-10%に制限推奨

## 発動頻度
- 月 1-2回 程度 (17ヶ月で32トレード)
- 「下落局面が到来したときだけ」発動する待ち伏せ型戦略

## 資金枠
- **¥500-1,000万** (独立枠)
- 10営業日保有 → 資金回転率 低 (月2回発動で2週間保有)
- 同時に最大2ポジション (新規と残り数日のもの重複許容) を想定

## 他戦略との相互作用

### vwap_morning_meanrevert (レーザーテック共通対象)
- vwap_morning_meanrevert は **イントラ (日中完結)**、本戦略は **スイング (10営業日)**
- **時間軸が完全に独立**、同時保有可能
- vwap_morning_meanrevert が逆張りShort (VWAPから+275bps上) を発動していても、スイング側のMA25サポートLongは独立管理

### orb_breakout_long
- ORB は半導体のディスコ/三井金属、本戦略はレーザーテック → 銘柄重複なし
- 独立運用

### 他 ON戦略
- 完全独立

## ストップロス詳細
- **-10% mode=low (日中Low到達で stop level 約定)**
- 逆指値 (成行) を板に置く
- 約定は stop level 固定ではなく成行に近い (slippage 1-2% 許容)
- 検証 (`analyses/20260422_lasertec_ma25/stop_loss.py`) で:
  - Stop -5% だと騙し下げで弾かれ 22% stop発動 → Sharpe 5.66 に悪化
  - Stop -7% / -10% が最適: Sharpe 7.59 / 7.68、stop率 3-6%

## 破綻条件 / 一時停止条件
- 3ヶ月連続マイナス → 一時停止・診断
- MA25 が明確な下降トレンド (5営業日前比で -3% 以上) に転じる → 新規エントリー停止
- Stop 発動率が 20% を超える3ヶ月連続 → 地合い悪化、一時停止
- レーザーテック単独の重大ニュース (業績下方修正、規制リスク、大口主要顧客喪失) → 即座にフラット化

## 既知のリスク
1. **銘柄集中**: 1銘柄依存、業績・規制・地合いで全てやられる可能性
2. **サンプル期間が短い** (17ヶ月): バックテスト期間内はレーザーテックの強い上昇相場 (14,000 → 35,000) があり、この上昇トレンドが効果を増幅している可能性
3. **トレンド依存**: MA25 が上昇中の条件は、下降トレンド相場では発動しないので戦略が休眠する
