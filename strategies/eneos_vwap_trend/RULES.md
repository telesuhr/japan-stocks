# 意思決定フローチャート — ENEOS VWAP Trend

---

## 9:30 — エントリー判定

```
START (9:30 JST)
  │
  ▼
① VWAP乖離を計算
   dev = (close_9:30 / VWAP_9:30 - 1) × 10000  [bps]
  │
  ▼
② |dev| ≥ 50bps か?
  │
  ├── No  → スキップ (skipped_reason=dev_below_threshold)
  │
  └── Yes ▼
       dev > 0 → Long (上乖離)
       dev < 0 → Short (下乖離)

③ 本日に決算発表があるか?
  │
  ├── Yes → スキップ (skipped_reason=earnings)
  │
  └── No  ▼

④ 市場全体が急変動中か?
   (日経225先物が ±3%超の動き)
  │
  ├── Yes → スキップ (skipped_reason=market_panic)
  │
  └── No  ▼

✅ 9:31〜: 次の1分バーのopenで成行エントリー
```

## 日中 — ホールド中

```
⑤ ポジション取得後、急変動が発生したか?
   (ENEOSが-3%超の急落 / +3%超の急騰)
  │
  ├── -3%超急落 (Longの場合) → 損切り検討（裁量）
  │
  ├── +3%超急騰 (Longの場合) → 利益確定 or ホールド継続（裁量）
  │
  └── 通常変動 → そのままホールド継続
```
