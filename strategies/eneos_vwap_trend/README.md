## 戦略構造

```
9:00      寄付き。VWAP計算開始（累積出来高加重平均）

9:30      VWAP乖離を確認
           └─ dev ≥ +50bps → Long  (株価がVWAPより0.5%上)
           └─ dev ≤ -50bps → Short (株価がVWAPより0.5%下)
           └─ |dev| < 50bps → スキップ

9:30直後   次の1分バーのopenで成行エントリー

15:20-15:30  引成（CLO）で決済
```

**シグナルと取引の関係**:
- 朝の勢いがVWAPに対して出ている日は、そのまま引けまで継続する傾向
- Long/Short 対称に機能するため、上下どちらの相場でも発動可能
- 純粋なイントラデイ戦略 — オーバーナイトリスクなし

## VWAP乖離の定義

```
VWAP_t = Σ(close_i × volume_i) / Σ(volume_i)  ← 9:00からの累積
dev_t   = (close_t / VWAP_t - 1) × 10000        ← bps単位

採用閾値: |dev| ≥ 50bps at 9:30 JST
```

volumeが0の場合は close×1 で代替。
