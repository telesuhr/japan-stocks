## Day N 15:20 — エントリー判定

```
START (Day N 15:20)
  │
  ▼
① LME銅 東京時間変化率 (JST 9:00 → 15:25) が +1.0% 以上か?
  │
  ├── No  → スキップ (skipped_reason=lme_below_threshold)
  │
  └── Yes ▼

② 本日 (Day N) は木曜日か?
  │
  ├── Yes → スキップ (skipped_reason=thursday)
  │
  └── No  ▼

③ Day N または Day N+1 に FOMC/米雇用統計/日銀会合があるか?
  │
  ├── Yes → サイズ半減 (各¥500万) or スキップ
  │
  └── No  ▼

④ コア5銘柄のうち Day N または Day N+1 が決算日の銘柄があるか?
  │
  ├── Yes → 該当銘柄のみ除外。残り銘柄数 < 3 なら全体スキップ
  │
  └── No  ▼

⑤ 日経225先物が Day N 15:20 時点で -2% 超急落中か?
  │
  ├── Yes → スキップ (skipped_reason=panic)
  │
  └── No  ▼

✅ 15:25-15:29: コア5銘柄 引成Long 発注
   15:30 大引け: 約定
```

## ポジションサイジング

```
標準: 各銘柄 ¥1,000万、5銘柄合計 ¥5,000万

例外:
  銘柄除外あり (N=4)  → 各¥1,000万のまま (合計¥4,000万)
  銘柄除外あり (N=3)  → 各¥1,000万のまま (合計¥3,000万)
  銘柄除外あり (N<3)  → 全体スキップ
  マクロイベント前夜  → 各¥500万に半減
  直近3ヶ月連続負け   → 各¥500万に半減 (暫定)
```

## リスク上限

```
月次: 月間 Net P&L が -¥200万 到達 → 月内新規エントリー停止
年次: 年間 Net P&L が -¥500万 到達 → 戦略停止・再検証
```

## 記録必須事項

- LME 東京オープン価格 (JST 9:00 or 10:00)
- LME 15:25 価格・変化率
- エントリー曜日・除外フラグ
- 各銘柄 entry/exit 価格・株数・bps
- Gross / Net P&L
- スキップ時は skipped_reason を必ず記録
