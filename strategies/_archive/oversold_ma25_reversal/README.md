# 過売り反発 (Oversold MA25 Reversal) Long 戦略

**型**: スイング Long (5営業日保有)
**バスケット**: プライム全銘柄 (除外: 医薬品)
**昇格元**: [`analyses/20260515_oversold_ma25_validation/`](../../analyses/20260515_oversold_ma25_validation/)

---

## 1. 戦略の核

### コンセプト
**平均回帰アノマリー**: 25日移動平均から -20% 以上下に大幅乖離した銘柄は、
平均回帰メカニズムにより5営業日で反発する傾向が極めて強い。

### 経済的根拠
- **過剰反応の修正**: 悪材料に対する過剰反応 (パニック売り) 後の冷静さ回復
- **空売り買い戻し**: -20%乖離銘柄は空売りが積み上がっており、買い戻し圧力
- **バリュー投資家の参入**: 「割安」と見て買いを入れる中長期投資家
- **テクニカル底**: -2σ以下の極端な水準に達した銘柄は機械的に反発しやすい
- **米国市場のempirically観測されている mean reversion strategy** と整合的

### シグナル
**Day N 15:25** に以下を満たした銘柄を抽出:
- 当日終値が **25日移動平均 × 80% 以下** (= MA25から -20% 以上下に乖離)
- 当日売買代金 ≥ **10億円** (流動性確保)
- セクター除外: 医薬品 (バックテストで反発弱い)
- プライム市場のみ

### エントリー / 出口
| 項目 | 内容 |
|------|------|
| エントリー | Day N+1 **09:00 寄成 Long** |
| 出口 | Day N+5 **15:30 引成 Sell** |
| **損切 (SL)** | **-7%** (推奨) または **なし** (より高Sharpe) |
| 利食 (TP) | **なし** |
| ポジションサイズ | ¥1,000,000 / 銘柄 |
| 同時保有数上限 | **10銘柄** (vol乖離深い順) |

## 6. 運用ルール

### 日次フロー
1. **15:25** Day N 引け直前
   `python3 signal_check.py` 実行
2. シグナル判定:
   - GO → Day N+1 09:00 寄成 Long で発注
   - SKIP → ノートレード
3. **同時に**: 5営業日前 (Day N-5) に建てたポジションを 15:30 引成 Sell

### 採択優先順位
1. 推奨セクター (銀行・輸送機・機械・電気機器・化学等) を優先
2. その中で MA25乖離が深い順
3. 同時保有 10銘柄まで

### SL運用
- 寄成 Buy 約定後、**約定価格 × 0.93** を SL ラインに設定
- intraday で SLラインに到達したら即時成行 Sell
- 約定価格 × 0.93 = -7% (バックテストでバランス最適)

### 月次レビュー
- 直近1ヶ月の実績集計
- net mean > 0、勝率 > 50% を維持確認
- **2026年弱化傾向** に注意

## 7. ⚠️ 重要な注意事項

### 2026年の性能反転
バックテストで 2026年 Sharpe -0.58 (N=245、4ヶ月のみ)。
- **要因仮説**: 円安一巡・米利下げ織込み・地政学リスク変化
- **対策**: 月次でSharpe<0なら ポジションサイズ半減
- **戦略停止**: 2ヶ月連続でnet<0、または6ヶ月Sharpe<0.5 → 一時停止

### 流動性リスク
- 大幅下落銘柄は流動性が低下している可能性
- 売買代金10億+のフィルタで概ね回避できているが、寄付スリッページに注意

### 生存バイアス
- 上場廃止リスクのある銘柄が含まれる可能性
- MBO・TOB情報を別途チェック
