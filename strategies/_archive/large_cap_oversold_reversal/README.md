**型**: スイング Long (5営業日保有)
**バスケット**: TOPIX Core30 + Large70 = 100銘柄 (超大型・大型のみ)
**昇格元**: [`analyses/20260515_large_cap_reversal_validation/`](../../analyses/20260515_large_cap_reversal_validation/)

## 1. 戦略の核

### コンセプト
**大型株の急落後の反発を捕る** — 機関投資家のリバランス買い、バリュー投資家の参入、空売り買い戻しが集中する局面を狙う。

### 既存 `oversold_ma25_reversal` との違い

| 項目 | oversold_ma25_reversal (既存) | **large_cap_oversold (本戦略)** |
|------|------|------|
| ユニバース | プライム全銘柄 (売買代金10億+) | **TOPIX Core30+Large70 限定** |
| 銘柄数 | 約1,500 | **100** |
| 主シグナル | MA25乖離 ≤ -20% | **20日下落率 ≤ -20%** |
| 流動性リスク | 中型・小型混在 | **超大型のみ、極めて高い** |
| 上場廃止リスク | 一部あり | **ほぼゼロ** |
| **2026年Sharpe** | **-0.58 ⚠** | **-0.31 (Core30のみ +0.83)** |
| Sharpe (全期間) | +3.21 | **+2.92** |

→ **2026年弱化を緩和** & **流動性・倒産リスクを完全排除**

### 経済的根拠
- **機関投資家リバランス**: TOPIX/Nikkei225組入の大型株は急落後に機械的買いが入る
- **ベンチマーク連動買い**: パッシブファンドの定期買い入れ
- **バリュー投資家の参入**: 大型優良株の「安値」と認識される
- **空売り買い戻し**: 大型株は空売り残が積み上がりやすく、リバ時のショートカバーで急騰

### シグナル
**Day N 15:25** に以下を満たした銘柄を抽出:
- TOPIX scale_cat ∈ {Core30, Large70}
- **20日下落率 ≤ -20%** (主シグナル) または **MA25乖離 ≤ -15%** (補助)
- 当日売買代金 ≥ **30億円** (大型株なら通常クリア)
- 除外セクター: 精密機器 (バックテストで負Sharpe)

| 項目 | 内容 |
|------|------|
| エントリー | Day N+1 **09:00 寄成 Long** |

| 同時保有数上限 | **8銘柄** |
| **採択優先順位** | **Core30 > 推奨セクター > 下落深い順** |

## 5. 運用ルール

### 日次フロー
1. **15:25** Day N 引け直前
   `python3 signal_check.py` 実行
2. シグナル判定:
   - GO → Day N+1 09:00 寄成 Long で発注
   - SKIP → ノートレード
3. **同時に**: 5営業日前 (Day N-5) に建てたポジションを 15:30 引成 Sell

### 採択優先順位
1. **Core30** (超大型31銘柄) を最優先
2. その中で**推奨セクター** (卸売・保険・非鉄・銀行・輸送機・電気機器・機械)
3. その中で **20日下落率の深い順**
4. 同時保有 8銘柄まで

### 月次レビュー
- 直近1ヶ月の実績集計
- Core30実績 vs Large70実績を分離して評価
- 2026年弱化傾向 (Large70) の継続性をモニター

## 6. ⚠️ 重要な注意事項

### 2026年の規模間乖離
- **Core30 (超大型): +0.83 (プラス維持)** ← 引き続き有効
- **Large70 (大型): -0.69** ← 一時的弱化、要監視
- 月次でLarge70がnet<0が続く場合 → **Core30限定に絞る**

### 既存戦略との重複
**oversold_ma25_reversal (既存)** と同じパターン (過売り反発) のため、同銘柄シグナル発生の可能性大。

**運用ルール**: 同銘柄シグナルが両戦略で発生した場合:
- 大型 (Core30/Large70) → **本戦略を優先**
- 中小型 (Mid400以下) → oversold_ma25_reversal を使用

両戦略を**併用**することで、サイズ階層の網羅性を確保。

### 流動性は問題なし
- 全銘柄が売買代金30億+ をクリア → ¥1000万/銘柄でも問題なし
- スケールアップ可能 (1ポジ ¥500万 × 5銘柄 = ¥2,500万まで実用可)
