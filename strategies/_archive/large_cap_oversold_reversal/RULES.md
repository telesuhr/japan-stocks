# 意思決定フローチャート — 大型株 過売り反発 Long 戦略

## 🟢 Day N 15:25 — 翌日エントリー判定

```
START (Day N 15:25)
  │
  ▼
① signal_check.py 実行
   - TOPIX Core30+Large70 (100銘柄)
   - 20日下落 ≤ -20% or MA25乖離 ≤ -15%
   - 売買代金 ≥ 30億
  │
  ▼
② シグナル ≥ 1 か?
  ├── No  → SKIP
  └── Yes ▼
③ 戦略停止フラグなし? (2026年Large70弱化チェック)
  ├── あり → SKIP
  └── なし ▼
④ 現在保有 < 8銘柄?
  ├── No  → SKIP
  └── Yes ▼
⑤ Day N+1, N+2 にFOMC/日銀会合なし?
  ├── あり → サイズ半減 or SKIP
  └── なし ▼
⑥ 採択ロジック:
   優先順位: Core30 > 推奨セクター > 下落深い順
   N銘柄採択 = min(8 - 現在保有, シグナル銘柄数)
  │
  ▼
GO (Day N+1 09:00 寄成 Long × ¥100万/銘柄)
```

## 🟡 Day N+1 09:00 — 寄成発注

- ① 寄付直後にギャップダウン -5%超 → 該当銘柄キャンセル
- ② 09:00 寄成 Buy 発注
- ③ trade_log に entry_date, entry_price 記録

## 🛑 保有中 (Day N+1 〜 Day N+5)

- SLなし運用が基本 (バックテストで最強)
- ただし **-15% 以上のドローダウン**なら手動Exit検討
- MBO/TOB/上場廃止リスク発表 → 即時成行Exit (大型なので可能性低い)

## 🔴 Day N+5 15:30 — 引成決済

- 全該当ポジションを引成 Sell
- trade_log に exit_date, exit_price, pnl 記録

## ⚠️ 戦略停止条件

1. 直近6ヶ月Sharpe < 0.5
2. **Large70 のみ 2ヶ月連続net<0** → **Core30限定に縮小** (戦略終了ではない)
3. Core30 含む全体で2ヶ月連続net<0 → **戦略一時停止**
4. 5トレード連続-5%以上損失
5. 暴落相場 (TOPIX 月次 -10%超)

## 📋 他戦略との連携

### oversold_ma25_reversal (既存) との関係
**両戦略は同じ過売り反発パターン**を使うが、ユニバースが異なる:
- 本戦略 (large_cap): Core30+Large70 (100銘柄)
- 既存 (oversold_ma25): 全プライム (Mid400以下も含む)

→ **重複時のルール**:
- 同銘柄が両戦略で出る場合: 本戦略を優先 (流動性・倒産リスク低)
- 異なる銘柄: 両方稼働可

### bank_absorption との関係
銀行株で重複可能性あり (本戦略の銀行業は推奨セクター):
- bank_absorption は出来高吸収パターン
- 本戦略は20日下落率パターン
- 異なるロジックなので**両方稼働可**

### 他のスイング戦略
- earnings_pead / pre_earnings_drift: 完全独立
- lasertec_ma25_support: レーザーテックは Mid400 なので本戦略の対象外
