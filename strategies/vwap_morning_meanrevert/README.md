## エッジの源泉

**仮説**: 機関投資家のVWAP執行アルゴリズムが「VWAPからの乖離」に対して引力として働く。
前場で day-VWAP から 275bps 以上乖離した銘柄は、後場以降に VWAP 方向へ回帰する圧力が生じる。

**なぜ前場だけ効くか**:
- 前場で大きく乖離 = 朝の寄付材料による過剰反応
- 後場にかけて VWAP 到達を目標とする発注が増える
- 13時台以降のエントリーは回帰の時間不足 or ノイズ

**なぜ早期決済は負けるか (11:25 exit は Sharpe ≤ -2)**:
- VWAP回帰は**数時間スパン** (速攻では戻らない)
- 引け間近の機関発注で最終的に収束する傾向

## 実運用ルール

### エントリー条件 (各銘柄独立に判定)
1. 当日 9:00 から day-VWAP を累積計算
   - `VWAP_t = Σ(close × volume) / Σ(volume)` (9:00からの累積)
   - `乖離(bps) = (close / VWAP - 1) × 10000`
2. 10:00 〜 11:30 の各 1分バーで \|乖離\| を監視
3. 初めて \|乖離\| ≥ 275bps を観測した bar で:
   - 乖離 > +275bps (VWAP より上) → **Short 成行**
   - 乖離 < -275bps (VWAP より下) → **Long 成行**
4. その銘柄は当日追加エントリーしない (1日1回ルール)

### エグジット
- **15:25 引成** (一律、例外なし)
- 早期決済は Sharpe を大きく毀損 (11:25 exit = Sharpe-2 以下) → **禁止**

### 銘柄構成
- 3銘柄等加重 (TEL / ディスコ / レーザー)
- 各銘柄独立に判定、複数同時発生は全て建てる
- 6503三菱電機・6857アドバン・非鉄銘柄は**エッジ不安定なため除外**

### サイズ
- 1銘柄 ¥300-500万 (信用余力次第で増額)
- 最大建玉 = 3銘柄全発動時で ¥900-1,500万

## リスクと注意点

### 1. シグナル頻度のレジーム依存
2026-03以降、市場ボラ上昇で trigger_rate (\|dev\|≥200bps) が 2-5% → 10-15% に急増。
本戦略の275bps閾値でもシグナルが急増する可能性あり。
→ **月間取引上限 (例: 10回/月)** を設けて過剰発注を防ぐ。

### 2. 高ボラ日は逆張り失敗リスク
当日 9:00-10:00 レンジが銘柄の上位1/3に入る日 (high regime) は Sharpe+0.63 と弱化。
→ 実運用時は **first-hour range が過去20日平均の2倍超** の日は発動見送りも検討。

### 3. N=51 は薄い
月間2-5回 × 1.5年 = 30-90回程度。統計的有意ではあるが、ペーパートレード継続で検証必須。

### 4. VWAPの定義
本戦略は **「9:00からの累積 day-VWAP」** を使用 (前日持ち越し無効)。
broker提供のVWAPは計算方法が異なる場合があるため、**自前計算推奨**。

### 5. Stop の非設定
本バックテストでは Stop を設定していない (15:25までホールド)。
想定最悪ケース: 乖離がさらに拡大し続けて引け時点で -500bps 以上の損失 → 実際に H1 で発生あり。
→ 実運用では **-400bps で強制決済** を追加検討 (バックテスト影響は軽微)。
