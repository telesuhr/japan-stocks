# pair_portfolio — 18ペア統合ペアトレード戦略

## 概要
日本株 18 ペアの日次 Z-score 平均回帰を **Equal-Weight ポートフォリオ化**することで、個別ペアの弱いエッジを相関ゼロで √N 増幅する統計的アービトラージ戦略。

## パフォーマンス (11年バックテスト: 2015-2026)

| 指標 | 値 |
|---|---|
| **年率 Sharpe** | **+1.37** |
| **t-stat** | **+4.55** |
| H1 / H2 Sharpe | +1.36 / +1.39 (両期安定) |
| 累積 PnL | +3,400 bps (34%) |
| Max Drawdown | 262 bps |
| **年次プラス率** | **11/11 年 全年プラス** |
| ペア間平均 \|ρ\| | 0.008 (ほぼ無相関) |
| 年間取引回数 | 〜70-80 回/ペア × 18 = 〜1,400 往復 |

## シグナル仕様

### エントリー条件
- ペアごとに個別最適化された entry_z (1.5 / 2.0 / 2.5) と z_window (20 / 40 / 60)
- 毎日 15:30 引け後 (または翌朝 07:00-08:30) に `signal_check.py` 実行
- `|Z| ≥ entry_z` が成立したペアを翌寄成で発注
  - **Z > +entry_z**: Spread Short = p1 Short + β×p2 Long
  - **Z < -entry_z**: Spread Long  = p1 Long  + β×p2 Short

### エグジット条件 (毎日引け後チェック)
1. `|Z| < exit_z` (ペア別 0.4〜0.6) → **MR exit**
2. `|Z| > 4.0` → **Stop exit**
3. 保有 `MaxHold` 日 (ペア別 10/20/30) 経過 → **Time exit**

### コスト前提
- 片側 4bps × 2 銘柄 = **8bps/往復** (ペア)
- 分足実行では 16bps に悪化するため **日次執行限定**

## ポジションサイジング

- 各ペア: 1/18 の等配分
- グロスノーショナルは `(1 + |β|) × 資金/18`
- **推奨総資金**: ¥3,000-5,000 万 (各ペア ¥170-280 万)
- **レバ 3-5 倍**で年率 ~10-15% (Sharpe 維持、MDD ~1,000-1,300 bps)

## 他戦略との関係

- ON 戦略 (lme_on_copper / topix_overnight) とは **独立資金枠**
- イントラ戦略 (eneos_vwap_trend / sox_overnight_short / vwap_morning_meanrevert / orb_breakout_long) とも独立
- 日次スプレッド MR なので、方向性戦略とは相関が極めて低い想定 (後述の相関分析で確認)
