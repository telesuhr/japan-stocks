# 日次オペレーションチェックリスト — 銀行株 出来高吸収逆張り戦略

---

## 🌅 Day N 15:35 — シグナル判定

### 前準備
- [ ] DB同期完了確認（`stocks_daily` の最新日が当日になっているか）
- [ ] ホワイトリスト最新版確認（whitelist.csv）

### 実行
```bash
cd ~/claude-code/japan-stocks
python3 strategies/bank_absorption/signal_check.py
```

### 判定結果ごとの行動

#### ✅ GO の場合（シグナル発生）
- [ ] 表示された採択銘柄を確認（最大3銘柄）
- [ ] 各銘柄のニュース・決算予定を確認
  - JQuants `earnings_calendar` または取引所サイトでチェック
  - Day N+1 〜 Day N+5 に決算予定 → 該当銘柄除外
- [ ] 配当落ち日チェック（Day N+1 〜 Day N+5）
- [ ] FOMC / 日銀会合カレンダー確認（Day N+1, N+2）
- [ ] 該当銘柄が現在ホールド中でないか確認（重複防止）
- [ ] **発注準備**: Day N+1 09:00 寄成 Buy ¥100万/銘柄

#### ❌ SKIP の場合
- [ ] ノートレード確認
- [ ] signals_log.csv に記録されていることを確認

## 🟢 Day N+1 09:00 — 寄成発注

### 寄付き前 (08:55-08:59)
- [ ] 採択銘柄の気配値確認
- [ ] 制限値幅 ±10%超 or ストップ高/安気配 → 該当銘柄キャンセル
- [ ] 大型ニュース（決算リーク・買収等）でギャップ±5%超 → 該当銘柄キャンセル

### 09:00 発注
- [ ] 各銘柄を **寄成 Buy ¥100万** で発注
- [ ] 約定確認（09:05 まで）
- [ ] 不約定銘柄は当日キャンセル → 翌日以降の再発注はしない

### 記録
- [ ] trade_log.csv に以下を記録
  - signal_date / entry_date / code / name
  - 寄成発注価格 / 実約定価格 / 約定数量
  - シグナル時のvol_ratio・day_ret

---

## 📊 保有中 (Day N+1 〜 Day N+5)

### 毎日 15:35 引け後
- [ ] 各保有銘柄の終値ベース評価損益を記録
- [ ] **-10%以上のドローダウン** → 翌日寄付きで手動Exit検討
- [ ] **+15%以上の利益** → 通常通り Day N+5 まで保有（テイクプロフィット設けない）

### 異常事態
- [ ] 銘柄に上場廃止・MBO・TOB等の発表 → 即座に Day翌日寄成 Exit
- [ ] 銀行業界全体への規制発表 → 全ポジション緊急Exit検討

---

## 🔴 Day N+5 15:30 — 引成決済

### 15:25-15:29
- [ ] 各銘柄の引成 Sell 発注準備
- [ ] 板状況・気配値を確認

### 15:30 発注
- [ ] 全保有銘柄を **引成 Sell** で発注
- [ ] 約定確認

### 記録
- [ ] trade_log.csv に以下を記録
  - exit_date / exit_price / pnl / hold_days
  - hold_max_dd / hold_max_profit (参考)

## ⚠️ 緊急時対応
