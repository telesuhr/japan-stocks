# 銀行株 出来高吸収逆張り戦略

**型**: スイング Long (5営業日保有)  
**バスケット**: ホワイトリスト22銘柄（プライム上場・銀行業セクター）  
**昇格元**: [`analyses/20260510_bank_absorption_strategy/`](../../analyses/20260510_bank_absorption_strategy/)

---

## 1. 戦略の核

### コンセプト
「**大商いで陰線**」は機関投資家による**売り吸収**の兆候。買い手が売り圧力を吸収しきった後にリバウンドが起きる確率が高いという仮説に基づく。

### シグナル
日次引け時点で以下を**全て**満たした銘柄：
- 当日出来高 ≥ 過去20日平均 × **1.5倍**
- 当日リターン (`adj_close / adj_open - 1`) **< 0%** （陰線）
- 当日売買代金 ≥ **10億円**
- 銘柄が **[whitelist.csv](whitelist.csv)** に含まれる

### エントリー / 出口
| 項目 | 内容 |
|------|------|
| エントリー | Day N+1 09:00 寄成Long |
| 出口 | Day N+5（5営業日後）15:30 引成 |
| SL/TP | **なし**（バックテストで悪化が確認された）|
| ポジションサイズ | ¥1,000,000 / 銘柄 |
| 同時保有数上限 | **3銘柄**（vol_ratio降順で選定）|

## 4. 運用ルール

### 日次フロー
1. **15:35** Day N 引け後  
   `python3 signal_check.py` 実行
2. シグナル判定結果に基づき:
   - GO → Day N+1 09:00 寄成発注（最大3銘柄）
   - SKIP → ノートレード
3. **Day N+5 15:30** 全ポジション引成決済

### 重複時のルール
- ホワイトリスト銘柄が**3超**シグナル発生時 → vol_ratio降順で上位3銘柄
- ON系戦略（lme_on_copper / topix_overnight）との**重複可**（資金枠が独立）

### 月次メンテナンス
- 毎月初: 直近1ヶ月の実績をログ集計
- 6ヶ月毎: ホワイトリスト再検証（過去24ヶ月でSharpe>1.0を確認）
- **2026年Sharpe<0.5に陥った場合**は戦略一時停止

## 5. 想定資金枠

- 通常運用: ¥3,000,000（3銘柄 × ¥100万）
- 年あたりエントリー機会: 約100件 → 月8〜10回
- 年率期待リターン: ~30〜50%（複利・1ポジ比例）
- 最大想定DD: -55万円（資金の約18%）

## 6. 限界・注意点

### 統計的限界
- N=771は基準1000未満（年300件の流入で約3年待つ必要あり）
- 2026年の性能低下が継続するか不明（金利環境変化の影響？）

### 実装上の注意
- **流動性**: 売買代金10億円フィルタ済みだが、マーケットインパクトは未検証
- **寄付きスリッページ**: 寄成発注のスリッページは2-5bpsを想定（コスト10bps計算に内包）
- **約定不可リスク**: ストップ高/安銘柄が混じる場合あり（要除外チェック）

### 拡張余地
- 他セクター（非鉄金属 Sharpe 1.94）への展開検討
- 1分足ベースの**前場引け時点**で先行シグナル化
- 信用残・空売り比率を追加フィルタ化
